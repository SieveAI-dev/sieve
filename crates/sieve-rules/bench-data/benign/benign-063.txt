Solana DeFi protocol development notes:

Common patterns in Solana DeFi:
- Token swaps via AMM (Automated Market Maker)
- Liquidity pools
- Yield farming contracts
- Cross-program invocations (CPI)

For testing, use the localnet cluster:
```bash
solana-test-validator
solana config set --url localhost
```

Test transactions should use devnet keypairs, never mainnet private keys.
