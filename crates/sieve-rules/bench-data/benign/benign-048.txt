Fish shell configuration guide:

Fish is a user-friendly interactive shell. Configuration file is located at:
~/.config/fish/config.fish

Common configuration options:
```fish
# Add to PATH
set -gx PATH $HOME/.local/bin $PATH

# Set default editor
set -gx EDITOR nvim

# Custom abbreviations
abbr -a gc 'git commit'
abbr -a gp 'git push'
```

Note: Editing this file changes your fish shell environment.
