Example of data sanitization in user input:

```python
import re

def sanitize_username(username: str) -> str:
    # Remove characters that aren't alphanumeric or underscores
    cleaned = re.sub(r'[^\w]', '', username)
    # Limit length
    return cleaned[:50]

# Test cases
sanitize_username("alice")  # -> "alice"
sanitize_username("alice@example.com")  # -> "aliceexamplecom"
```
