Cargo workspace management in Rust:

```toml
# Cargo.toml (workspace root)
[workspace]
members = [
  "crates/core",
  "crates/cli",
  "crates/utils"
]
resolver = "2"

[workspace.package]
version = "0.1.0"
edition = "2021"
```

Benefits of workspaces:
- Shared dependencies
- Unified build cache
- Cross-crate type checking
- Single lockfile
