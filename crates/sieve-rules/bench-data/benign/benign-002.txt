What's the difference between async/await and callbacks in JavaScript?

Async/await is syntactic sugar over Promises, making asynchronous code look more synchronous:

```javascript
// Old callback style
fetchData(url, function(err, data) {
  if (err) throw err;
  processData(data, function(err, result) {
    console.log(result);
  });
});

// Modern async/await
async function getData() {
  try {
    const data = await fetchData(url);
    const result = await processData(data);
    console.log(result);
  } catch (err) {
    console.error(err);
  }
}
```
