Setting up Prometheus monitoring:

```yaml
# prometheus.yml
global:
  scrape_interval: 15s

scrape_configs:
  - job_name: 'myapp'
    static_configs:
      - targets: ['localhost:8080']
    metrics_path: '/metrics'
```

Key metrics to track:
- HTTP request duration (histogram)
- Error rate (counter)
- Active connections (gauge)
- Memory usage (gauge)
