Firewall configuration documentation:

Current firewall rules (iptables):
```
-A INPUT -m state --state ESTABLISHED,RELATED -j ACCEPT
-A INPUT -p tcp --dport 22 -j ACCEPT   # SSH
-A INPUT -p tcp --dport 80 -j ACCEPT   # HTTP
-A INPUT -p tcp --dport 443 -j ACCEPT  # HTTPS
-A INPUT -j DROP
```

All other inbound traffic is dropped by default.
Note: Rule changes require sudo access and should be documented in the change log.
