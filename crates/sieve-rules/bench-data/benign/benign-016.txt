How to set up SSH key-based authentication:

First, generate an SSH key pair:
```bash
ssh-keygen -t ed25519 -C "your_email@example.com"
```

This creates two files:
- ~/.ssh/id_ed25519 (private key - keep this secret!)
- ~/.ssh/id_ed25519.pub (public key - this is safe to share)

Then add your public key to the server's authorized_keys file:
```bash
ssh-copy-id user@server.example.com
```
