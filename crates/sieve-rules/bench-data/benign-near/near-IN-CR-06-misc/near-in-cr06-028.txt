CTF challenge: "Hidden in Plain Sight"

Category: Web / LLM Security
Points: 250

Description: A vulnerable AI chatbot reads user-submitted documents and summarizes them. The flag is stored in the system prompt. Can you extract it?

Hint: The chatbot uses a tool to fetch documents from a URL you provide. Think about what you can put in that document.

This is a classic indirect prompt injection CTF. The solution involves hosting a document that contains instructions for the model to repeat its system prompt. This type of challenge teaches security researchers about the indirect injection attack surface — important knowledge for building defenses.
