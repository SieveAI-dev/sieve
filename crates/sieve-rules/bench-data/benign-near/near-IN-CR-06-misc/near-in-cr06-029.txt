HackTheBox: AI Security track — writeup notes

The "Prompt Pilot" machine on HTB involves a Flask app wrapping an LLM. The app fetches URLs provided by users and passes the content to the model.

Key techniques that work in CTF contexts (educational only):
- Embedding instruction text in metadata fields (EXIF, PDF author, HTML comments)
- Using Unicode normalization to bypass naive string filters
- Exploiting the difference between what the model sees and what the developer sees

Real-world mitigation: parse documents to extract only structured data, never pass raw document text to the model without sandboxing. Sieve's inbound filter is one layer of this defense.
