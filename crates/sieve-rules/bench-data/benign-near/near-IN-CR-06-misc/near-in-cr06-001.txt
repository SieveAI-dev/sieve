Prompt injection is a class of attacks against LLM-based applications where an attacker embeds instructions inside untrusted content (web pages, documents, tool outputs) that cause the model to deviate from its intended behavior. This is distinct from traditional SQL injection but follows a similar principle: mixing code and data in a channel that lacks proper separation. OWASP LLM Top 10 lists prompt injection as LLM01, the top risk for production AI systems.
