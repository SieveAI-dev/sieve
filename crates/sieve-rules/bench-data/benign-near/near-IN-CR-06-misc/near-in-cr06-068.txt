Building a secure MCP tool for blockchain operations

Security checklist for blockchain MCP tools:

- [ ] Never accept private keys as tool input parameters
- [ ] Use hardware wallet or OS keychain for key storage
- [ ] Display transaction details to user before signing
- [ ] Implement transaction simulation (check state diff before signing)
- [ ] Rate limit signing operations
- [ ] Log all signing requests with full parameters
- [ ] Require explicit user confirmation for amounts above threshold
- [ ] Validate recipient addresses are on an allowlist for automated transfers
- [ ] Reject transactions with unusual gas prices (MEV indicator)
- [ ] Implement daily transfer limits with manual override

IN-CR-05 catches unexpected signing calls. Add application-level controls on top.
