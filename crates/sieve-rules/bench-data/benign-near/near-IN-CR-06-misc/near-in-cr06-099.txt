Sieve open-source governance plan

Sieve is open-sourced under a permissive license. Design decisions about the open-source model:

- License: Apache-2.0 for code, CC BY-NC-SA 4.0 for docs
- Rule set: rules are open regardless of distribution channel
- Governance: maintained by the SieveAI team, open to outside contributors
- CVE process: security issues via GitHub Security Advisories (private disclosure)
- Release cadence: semantic versioning, releases tagged in git and signed via sigstore

Every design decision is documented in ADRs published in the repository.
