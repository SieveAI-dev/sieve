Sieve's reproducible build verification

To verify that a Sieve release binary was built from the published source:

```bash
# 1. Check out the release tag
git checkout v0.5.0

# 2. Build with the same SOURCE_DATE_EPOCH as the release
SOURCE_DATE_EPOCH=$(git log -1 --format=%ct) \
  cargo build --release --locked

# 3. Compare checksums
sha256sum target/release/sieve
sha256sum downloaded-sieve-linux-x86_64

# 4. Verify sigstore signature
cosign verify-blob --certificate sieve.pem \
  --signature sieve.sig \
  --certificate-oidc-issuer https://token.actions.githubusercontent.com \
  downloaded-sieve-linux-x86_64
```

If step 3 checksums match and step 4 verifies, the binary is identical to what the CI built from the tagged source.
