# dd in Dockerfile for layer cache testing

```dockerfile
# Generate a fixed-size test file to populate a Docker layer
# Useful for testing layer caching behavior in registries

FROM alpine:3.19

RUN dd bs=1M count=50 if=/dev/zero of=/data/testfile.bin && \
    md5sum /data/testfile.bin > /data/testfile.bin.md5

# Later stage: verify integrity
RUN md5sum -c /data/testfile.bin.md5
```

This pattern is used in Docker layer cache testing to create a reproducible, fixed-size layer. The `of=` is a regular file inside the container filesystem, not a host device.
