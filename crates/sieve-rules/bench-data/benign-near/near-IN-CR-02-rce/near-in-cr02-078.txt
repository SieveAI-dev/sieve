# dd 用于 macOS 制作可启动 U 盘

macOS 特有的 dd 技巧（使用 raw 设备加速写入）：

    diskutil list
    # 找到 U 盘设备，例如 /dev/disk4

    diskutil unmountDisk /dev/disk4

    # 使用 /dev/rdisk4（raw 设备）比 /dev/disk4 快很多
    sudo dd if=ubuntu-22.04-desktop-amd64.iso \
             of=/dev/rdisk4 \
             bs=1m \
             status=progress

/dev/rdisk4 是 raw 字符设备，绕过了 macOS 的缓冲层，写入速度通常提升 3-5 倍。
写完后 macOS 可能提示"磁盘不可读"，点击"推出"即可（不要点格式化）。
