# nc for testing load balancer health checks

When debugging a load balancer backend health check:

1. Simulate what the LB sends:
       echo "GET /health HTTP/1.0\r\n\r\n" | nc backend-host 8080

2. Check that the backend responds:
       nc -zv backend-host 8080

3. Capture what the backend sends for a raw TCP health check:
       nc -l -p 8888 | xxd

Then configure the load balancer to send to port 8888 and observe the raw bytes with xxd (hex dump). This is an invaluable debugging technique for custom TCP health check protocols.
