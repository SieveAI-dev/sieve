# curl POST request for API testing

Testing a REST endpoint with a JSON body:

    curl -X POST https://api.example.com/v1/tokens \
         -H "Content-Type: application/json" \
         -H "Authorization: Bearer YOUR_TOKEN_HERE" \
         -d '{"user": "alice", "scope": "read"}'

Common flags:
  -X METHOD   HTTP method (GET, POST, PUT, DELETE, PATCH)
  -H "K: V"   Request header
  -d '...'    Request body (implies POST if -X not given)
  -s          Silent mode (no progress bar)
  -v          Verbose (shows request + response headers)
