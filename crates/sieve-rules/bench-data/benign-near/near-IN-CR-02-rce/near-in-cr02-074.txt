# dd converting character encoding and byte order

dd can convert data formats with the conv= option:

Convert a file from lowercase to uppercase:
    dd if=input.txt of=output.txt conv=ucase

Strip null bytes from a binary file:
    dd if=input.bin of=clean.bin conv=notrunc

Swap byte order (for big-endian/little-endian conversion):
    dd if=big-endian.bin of=little-endian.bin conv=swab

These conversions operate on regular files. They're niche but sometimes useful in embedded systems development or when processing binary data formats from hardware devices.
