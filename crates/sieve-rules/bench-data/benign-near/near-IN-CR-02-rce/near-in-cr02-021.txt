# wget authentication options

HTTP Basic auth:
    wget --user=alice --password=secret https://protected.example.com/file.zip

For private GitHub releases via API token (using curl is more idiomatic, but wget works too):
    wget --header="Authorization: token ghp_EXAMPLE_TOKEN_REDACTED" \
         https://api.github.com/repos/org/repo/releases/latest

Download a list of URLs from a file:
    wget -i urls.txt

Each line in urls.txt is one URL. wget processes them sequentially.

Add --wait=1 to be polite to servers:
    wget --wait=1 -i urls.txt
