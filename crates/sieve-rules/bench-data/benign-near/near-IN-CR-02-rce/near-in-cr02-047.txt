# eval "$(brew shellenv)" — Homebrew path setup

On Apple Silicon Macs, Homebrew installs to /opt/homebrew instead of /usr/local. The recommended way to add it to PATH:

    eval "$(/opt/homebrew/bin/brew shellenv)"

This line goes in ~/.zprofile (for zsh) or ~/.bash_profile (for bash). What it outputs:

    export HOMEBREW_PREFIX="/opt/homebrew";
    export HOMEBREW_CELLAR="/opt/homebrew/Cellar";
    export PATH="/opt/homebrew/bin:/opt/homebrew/sbin:$PATH";
    ...

The eval executes those export statements in the current shell session. The source is a local Homebrew binary, not the network. This is a standard part of Homebrew's post-install setup instructions.
