# curl cookies and session management

Save cookies to a file:
    curl -c cookies.txt https://example.com/login \
         -d "user=alice&pass=secret"

Reuse saved cookies:
    curl -b cookies.txt https://example.com/dashboard

View and send a specific cookie header:
    curl -H "Cookie: session=abc123; csrftoken=xyz" https://example.com/api

Cookie jar files are plain text (Netscape format). You can inspect them with any text editor. Never commit cookie files to version control since they contain active session tokens.
