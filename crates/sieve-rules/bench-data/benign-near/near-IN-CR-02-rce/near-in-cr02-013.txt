# curl output formatting with jq

Fetch and filter JSON in a single command (piping to jq, not to a shell):

    curl -s https://api.github.com/repos/rust-lang/rust/releases \
      | jq '.[0] | {tag_name, published_at, body}'

Extract a specific field:
    curl -s https://api.github.com/users/octocat | jq -r '.name'

The -r flag outputs raw strings (no JSON quotes). This is useful when storing API responses in shell variables:
    LATEST=$(curl -s https://api.example.com/version | jq -r '.tag')

jq is a data transformation tool, not a code executor.
