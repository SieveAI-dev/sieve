# wget basic usage

Download a file and save it locally:

    wget https://example.com/releases/app-1.0.tar.gz

By default wget saves to the current directory using the remote filename. To specify an output filename:

    wget -O myapp-latest.tar.gz https://example.com/releases/latest.tar.gz

To download without saving (print to stdout), use -O-:

    wget -O- https://api.example.com/status

The -O- pattern writes to stdout. This is useful for inspecting the content before deciding what to do with it—view it, not execute it.
