# dd for disk performance benchmarking

Write performance test (generates a 1 GB file of zeros):

    dd bs=1M count=1024 if=/dev/zero of=/tmp/bench-write.bin status=progress

Read performance test (reads back the file):
    dd bs=1M if=/tmp/bench-write.bin of=/dev/null status=progress

The output shows throughput in MB/s:
    1073741824 bytes (1.1 GB, 1.0 GiB) copied, 2.5 s, 430 MB/s

This is a standard I/O benchmark. The target is a regular file (/tmp/bench-write.bin) and /dev/null (a null sink), not a physical device. Safe to run as a regular user.
