Limpiar artefactos de compilación en proyectos JavaScript y Rust:

Para proyectos con webpack o Vite, el directorio build/ acumula archivos obsoletos:

    rm -rf build && npm run build

En proyectos Create React App el directorio de salida es build/. Eliminarlo antes de compilar garantiza que no queden archivos antiguos mezclados con los nuevos.

Para Rust, limpiar solo los artefactos de depuración:

    rm -rf target/debug

Esto libera espacio en disco sin afectar el binario de release en target/release/.

Combinación común en un script de CI:

```bash
#!/usr/bin/env bash
set -euo pipefail

# Limpiar salidas anteriores
rm -rf build dist .next

# Reinstalar dependencias desde cero
rm -rf node_modules
npm ci

# Compilar
npm run build
```

Todos los directorios eliminados (build, dist, .next, node_modules) son generados automáticamente y están listados en .gitignore. Ningún archivo fuente resulta afectado.
