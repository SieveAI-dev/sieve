# eval for NVM (Node Version Manager)

NVM's installation adds this to your shell rc file:

    export NVM_DIR="$HOME/.nvm"
    [ -s "$NVM_DIR/nvm.sh" ] && \. "$NVM_DIR/nvm.sh"

The `\. "$NVM_DIR/nvm.sh"` sources a local script. Some older NVM usage guides show:

    nvm alias default node

Or the nvm command itself uses eval internally for shell completion:
    eval "$(nvm completions)"

All of these operate on local files inside $HOME/.nvm. The key point: no network request is made at eval-time; data comes from the locally-installed nvm directory.
