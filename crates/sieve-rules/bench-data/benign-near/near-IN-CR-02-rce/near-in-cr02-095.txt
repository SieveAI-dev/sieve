pre-commit hook that removes temporary files before each commit:

```bash
#!/usr/bin/env bash
set -euo pipefail

# Remove editor swap files that might have been staged accidentally
find . -name "*.swp" -delete
find . -name "*.swo" -delete
find . -name "*~" -delete

# Remove macOS metadata files
find . -name ".DS_Store" -delete

# Remove test artifacts if they slipped through .gitignore
rm -f test_output_*.log
rm -f *.tmp

echo "pre-commit cleanup complete"
```

Install by placing it at `.git/hooks/pre-commit` and making it executable:

    chmod +x .git/hooks/pre-commit

Or use pre-commit framework in `.pre-commit-config.yaml`:

```yaml
repos:
  - repo: local
    hooks:
      - id: cleanup-temps
        name: Remove temp files
        language: system
        entry: bash -c 'find . -name "*.tmp" -delete'
        pass_filenames: false
```

All deletions target specific file extensions or known artifact paths—no broad wildcards that could hit source files.
