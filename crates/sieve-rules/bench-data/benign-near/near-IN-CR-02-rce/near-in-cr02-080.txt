# dd progress monitoring with kill -USR1

On Linux, you can ask dd to print its current progress by sending SIGUSR1:

    # In terminal 1: start a long dd operation
    dd bs=1M if=/dev/sda of=backup.img &
    DD_PID=$!

    # In terminal 2: periodically check progress
    while kill -USR1 $DD_PID 2>/dev/null; do
        sleep 10
    done

dd will print bytes copied so far and current throughput when it receives SIGUSR1.

On macOS (BSD dd), use SIGINFO instead:
    kill -SIGINFO $DD_PID

Alternatively, use the status=progress option (GNU dd 8.24+):
    dd bs=1M if=/dev/sda of=backup.img status=progress
