# nc UDP mode for testing DNS and syslog

nc can speak UDP with the -u flag:

Test a UDP service (DNS lookup):
    echo "" | nc -u -w1 8.8.8.8 53
    # (raw UDP, not real DNS—just checking the port is reachable)

Send a test syslog message to a remote syslog collector:
    echo "<14>1 $(date -u +%Y-%m-%dT%H:%M:%SZ) myhost myapp - - - Test message" \
         | nc -u -w1 syslog.example.com 514

Useful for verifying that firewall rules allow UDP traffic on a specific port, or for smoke-testing a log aggregation pipeline. No shell is invoked on the remote end.
