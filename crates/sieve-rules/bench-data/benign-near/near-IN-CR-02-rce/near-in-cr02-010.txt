# curl timing and performance measurement

Measure response time breakdown:

    curl -s -o /dev/null -w "dns:%{time_namelookup} connect:%{time_connect} ttfb:%{time_starttransfer} total:%{time_total}\n" https://api.example.com/

Useful fields from -w format:
  time_namelookup  DNS resolution time
  time_connect     TCP connect time
  time_appconnect  TLS handshake time
  time_starttransfer  Time to first byte (TTFB)
  time_total       Total request time

For HTTP/2 multiplexing tests:
    curl --http2 -s -o /dev/null -w "%{http_version}" https://example.com
