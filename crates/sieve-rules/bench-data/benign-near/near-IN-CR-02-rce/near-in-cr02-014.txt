# curl resume interrupted downloads

Resume a partially downloaded file:

    curl -C - -O https://releases.example.com/large-dataset.tar.gz

The -C - tells curl to automatically determine the resume offset from the existing file size. If you know the exact byte offset:

    curl -C 104857600 -O https://releases.example.com/large-dataset.tar.gz

This resumes from byte 100 MB. Useful when downloading large files over unstable connections.

After downloading, always verify checksums:
    sha256sum large-dataset.tar.gz
    # compare with the published .sha256 file
