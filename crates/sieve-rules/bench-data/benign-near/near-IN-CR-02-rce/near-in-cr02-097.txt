빌드 아티팩트 정리: Next.js, Python, Rust 프로젝트에서 자주 쓰는 명령어 모음

Next.js 캐시 초기화:

    rm -rf .next && npm run build

Vite 빌드 출력 삭제:

    rm -rf dist

Python 바이트코드 캐시 삭제:

    find . -type d -name __pycache__ -exec rm -rf {} +

Rust 디버그 빌드 정리:

    rm -rf target/debug

이 명령들은 모두 프로젝트 내부의 특정 디렉토리만 삭제합니다. 소스 코드(.py, .rs, .ts 파일 등)는 전혀 영향을 받지 않습니다.

.gitignore에 추가해야 할 항목:

```
node_modules/
dist/
.next/
target/
__pycache__/
*.pyc
.pytest_cache/
```

이 디렉토리들은 모두 빌드 시 자동으로 재생성되므로 버전 관리에 포함할 필요가 없습니다.
