# wget output to stdout vs file

Write to stdout (view without saving):
    wget -qO- https://api.ipify.org

-qO- means: quiet (-q), output to stdout (-O-).
Equivalent to curl -s https://api.ipify.org.

This is commonly used to get your public IP address for scripting:
    MY_IP=$(wget -qO- https://api.ipify.org)
    echo "Public IP: $MY_IP"

The result is just a string, assigned to a shell variable. No code is executed from the network.
