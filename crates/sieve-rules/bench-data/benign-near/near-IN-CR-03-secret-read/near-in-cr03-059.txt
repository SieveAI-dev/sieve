Solana bip39 seed phrase recovery

When you run `solana-keygen new`, the CLI displays a BIP-39 seed phrase.
You can recover the keypair from it:

    solana-keygen recover 'prompt:?key=0/0'
    # Prompts for the seed phrase interactively (doesn't appear in shell history)

    # Recover and write directly to file
    solana-keygen recover --outfile recovered.json 'prompt:?key=0/0'

Derivation path `0/0` means account 0, address index 0 — matching the
default keypair for most Solana wallets.

Important: solana-keygen validates the BIP-39 checksum on the seed phrase.
If any word is wrong or the checksum doesn't match, recovery fails immediately
with an error — this prevents you from setting up the wrong keypair silently.
