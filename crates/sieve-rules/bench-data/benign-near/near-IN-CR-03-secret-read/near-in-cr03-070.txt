macOS Keychain and iCloud Keychain sync

Items stored with kSecAttrSynchronizable = true sync across Apple devices
via iCloud Keychain. Items with kSecAttrSynchronizable = false (or
kSecAttrAccessibleWhenUnlockedThisDeviceOnly) stay on the current device.

For app development:
    # Store with sync (crosses devices — appropriate for user-facing credentials)
    let query: [String: Any] = [
        kSecAttrSynchronizable as String: kCFBooleanTrue!,
        ...
    ]

    # Store without sync (stays local — appropriate for device-specific keys)
    let query: [String: Any] = [
        kSecAttrSynchronizable as String: kCFBooleanFalse!,
        kSecAttrAccessible as String: kSecAttrAccessibleWhenUnlockedThisDeviceOnly,
        ...
    ]

For code signing certificates and private keys, always use non-synchronizable
items — signing keys are device-specific and should not roam.
