GPG key management basics: the ~/.gnupg directory

The `~/.gnupg/` directory stores all GnuPG data:
  pubring.kbx      — public keyring (friends' keys, your own public key)
  private-keys-v1.d/ — private key material (encrypted with your passphrase)
  trustdb.gpg      — web of trust database
  gpg.conf         — per-user configuration

Listing keys (public key info only — no private key exposure):
    gpg --list-keys              # shows all public keys
    gpg --list-keys user@example.com  # filter by UID

The `--list-keys` output shows key IDs, fingerprints, UIDs, and expiry dates.
Fingerprints are safe to share publicly — they identify keys, not secret them.

Set correct permissions (GPG enforces this):
    chmod 700 ~/.gnupg
    chmod 600 ~/.gnupg/*
