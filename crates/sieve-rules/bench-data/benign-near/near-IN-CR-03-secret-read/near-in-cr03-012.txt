ssh-agent usage: avoiding passphrase prompts

ssh-agent holds decrypted private keys in memory for the duration of your
session, so you type the passphrase only once:

    # Start the agent (usually done by your desktop session / .profile)
    eval "$(ssh-agent -s)"

    # Add your key — you will be prompted for the passphrase once
    ssh-add ~/.ssh/id_ed25519

    # List loaded keys
    ssh-add -l

    # Remove all keys from the agent
    ssh-add -D

On macOS, Keychain integration means the passphrase is stored in the login
keychain after the first unlock, so you survive reboots without re-entering it:

    ssh-add --apple-use-keychain ~/.ssh/id_ed25519
