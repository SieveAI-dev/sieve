GPG for git commit signing

    # Configure git to sign commits with your GPG key
    git config --global user.signingkey YOUR_KEY_FINGERPRINT
    git config --global commit.gpgsign true

    # Sign a single commit
    git commit -S -m "feat: add new feature"

    # Verify signatures in the log
    git log --show-signature

    # For GitHub: add your public key under Settings → SSH and GPG Keys
    gpg --armor --export user@example.com | pbcopy

GitHub shows a green "Verified" badge on signed commits, proving the commit
was made by the owner of the GPG key, not just someone with write access to
the repo.

gpg.conf settings that help with git:
    use-agent
    default-key YOUR_FINGERPRINT
