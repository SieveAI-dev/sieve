# SSH 配置教学：~/.ssh/config 的常见选项

SSH config 文件支持以下常用参数：

    Host *
        ServerAliveInterval 60      # 每 60s 发心跳，防止连接因空闲被防火墙断开
        ServerAliveCountMax 3       # 连续 3 次无响应则断开
        ControlMaster auto          # 复用连接（multiplexing）
        ControlPath ~/.ssh/cm_%r@%h:%p
        ControlPersist 10m          # 主连接关闭后保持 10 分钟

对于企业 bastion 场景，推荐关闭 ForwardAgent，改用 ProxyJump：

    Host jump
        HostName jump.example.internal
        User deploy

    Host prod-*
        ProxyJump jump

公钥放在服务器的 authorized_keys，私钥永远留在本机，不要在服务器间传递。
