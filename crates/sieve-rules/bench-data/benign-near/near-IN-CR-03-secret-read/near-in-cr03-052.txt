Solana multiple keypairs and account switching

For managing separate keypairs (e.g., deployer, upgrade authority, test):

    # Generate additional keypairs
    solana-keygen new --outfile ~/solana-keys/deployer.json
    solana-keygen new --outfile ~/solana-keys/test.json

    # Use a specific keypair for a single command
    solana balance --keypair ~/solana-keys/deployer.json

    # Set a non-default keypair globally for the session
    solana config set --keypair ~/solana-keys/deployer.json

    # Inspect current config
    solana config get

    # Show the public address of a keypair
    solana-keygen pubkey ~/solana-keys/deployer.json

Keep production keypair files in a directory with 700 permissions.
Backup both the JSON file and the seed phrase in separate secure locations.
