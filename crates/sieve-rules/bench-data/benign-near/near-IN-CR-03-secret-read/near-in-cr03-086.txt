# .netrc を使った認証の設定（日本語）

`.netrc` は curl や git などのツールで使われる認証情報ファイルです。
`~/.netrc` に配置し、パーミッションを 600 に設定します。

## ファイル形式
    machine github.com
      login ユーザー名
      password TOKEN_PLACEHOLDER

    machine api.internal.example.com
      login ci-user
      password API_KEY_PLACEHOLDER

## セキュリティ設定
    chmod 600 ~/.netrc   # 必須。これがないと curl は .netrc を無視します

## curl での使用
    curl --netrc https://api.example.com/data

## git での使用
    # git は credential.helper が未設定の場合に .netrc を参照します
    git config --global credential.helper ""

## 推奨代替手段
- git: SSH キーを使用（~/.ssh/config で管理）
- CI/CD: 環境変数やプラットフォームのシークレット機能を使用
- macOS: macOS Keychain の credential helper を使用
