Listing keystore accounts with geth

    # List all accounts in the default keystore
    geth account list

    # List from a custom keystore directory
    geth --keystore /path/to/keystore account list

    # Output format:
    # Account #0: {4f3d...af12} keystore:///home/user/.ethereum/keystore/UTC--...

The command shows the public address (checksummed hex) and the path to the
keystore file. This is read-only metadata — it does not expose private keys
or passphrases.

For Hardhat / Foundry development, use the local network's auto-generated
accounts instead of your real keystore. Keep production keystores off
development machines entirely.
