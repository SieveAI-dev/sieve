Storing certificates in the macOS Keychain

    # Import a PKCS#12 (.p12) certificate into login.keychain
    security import certificate.p12 -k ~/Library/Keychains/login.keychain-db -P "p12-passphrase"

    # Import a public certificate (no private key)
    security import certificate.cer -k ~/Library/Keychains/login.keychain-db

    # List certificates in the keychain
    security find-certificate -a | grep "labl"

    # Add a trusted root certificate (requires admin)
    sudo security add-trusted-cert -d -r trustRoot -k /Library/Keychains/System.keychain CA.cer

Code signing with Keychain-stored certificates (Xcode):
- The build system reads signing identities from the keychain automatically.
- For CI, add the certificate to a temporary keychain during the build,
  then delete it after. Never store the .p12 and its passphrase in the repo.
