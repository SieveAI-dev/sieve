# macOS Keychain コマンドラインリファレンス（日本語）

`security` コマンドで Keychain を操作する（書き込み操作のみ）:

## パスワードの追加（書き込み）
    security add-generic-password -a アカウント名 -s サービス名 -w パスワード

## インターネットパスワードの追加
    security add-internet-password -a user@example.com -s example.com -w secret

## 証明書のインポート
    security import cert.p12 -k login.keychain-db -P "p12のパスフレーズ"

## キーチェーン一覧の表示
    security list-keychains

## キーチェーン情報の表示
    security show-keychain-info ~/Library/Keychains/login.keychain-db

## ロック/アンロック
    security lock-keychain login.keychain
    security unlock-keychain login.keychain   # パスワードを求めるプロンプトが出る

パスワードの読み取りやエクスポート操作はスクリプトから行わないこと。
