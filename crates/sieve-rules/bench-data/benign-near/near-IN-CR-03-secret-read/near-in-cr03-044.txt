Hardware wallets as a keystore alternative

For significant value, hardware wallets (Ledger, Trezor) replace software
keystores entirely:

- The private key is generated and stored on the device; it never leaves.
- Signing happens on-device; the computer only sees the signed transaction.
- Even if your computer is compromised, the private key cannot be extracted.

Integration with common tools:
    # ethers.js v6 with Ledger
    import { LedgerSigner } from "@anders-t/ethers-ledger";
    const signer = new LedgerSigner(provider, "m/44'/60'/0'/0/0");

    # Foundry: no native Ledger support yet; use a dedicated signing script

For development and testing, use ephemeral test accounts generated by
Anvil (Foundry) or Hardhat — never use hardware wallet keys on testnets.
