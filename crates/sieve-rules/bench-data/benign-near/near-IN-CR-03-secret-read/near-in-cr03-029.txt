Troubleshooting AWS credential errors

Common error: "NoCredentialError: Unable to locate credentials"

Check (in order):
1. Is AWS_PROFILE set to the right profile?
   echo $AWS_PROFILE

2. Is the profile defined in config/credentials?
   aws configure list --profile yourprofile

3. For SSO: are tokens still valid?
   aws sso login --profile yourprofile

4. For EC2/Lambda: is the instance/execution role attached?
   curl http://169.254.169.254/latest/meta-data/iam/info

5. Verify which identity the SDK resolves to:
   aws sts get-caller-identity --profile yourprofile

The credentials file at ~/.aws/credentials is only one of many credential
sources — check all of them systematically before concluding it's a file issue.
