Boto3 session management in Python

Rather than relying on the global default profile, pass sessions explicitly:

    import boto3

    def get_s3_client(profile: str = "default") -> boto3.client:
        session = boto3.Session(profile_name=profile)
        return session.client("s3")

    # In tests, inject credentials via environment variables instead:
    # AWS_ACCESS_KEY_ID=testing AWS_SECRET_ACCESS_KEY=testing pytest

For unit tests, use moto to mock AWS services:

    import moto
    @moto.mock_s3
    def test_upload():
        s3 = boto3.client("s3", region_name="us-east-1")
        s3.create_bucket(Bucket="test-bucket")
        ...

This way tests never touch real credentials or real AWS infrastructure.
