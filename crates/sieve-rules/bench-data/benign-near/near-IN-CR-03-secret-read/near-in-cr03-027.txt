AWS SSO with IAM Identity Center

Modern AWS setup replaces static credentials with SSO:

    # Configure once:
    aws configure sso
    # SSO session name: my-sso
    # SSO start URL: https://myorg.awsapps.com/start
    # SSO region: us-east-1
    # ...follow browser prompts...

    # Login (refreshes temporary credentials automatically):
    aws sso login --profile dev-account

    # Use normally:
    aws s3 ls --profile dev-account

Credentials are stored as short-lived tokens in ~/.aws/sso/cache/ —
not as long-term keys in ~/.aws/credentials. They expire automatically
and cannot be used if stolen after expiry.

For multiple accounts, configure one profile per account in ~/.aws/config
and switch with AWS_PROFILE or --profile flag.
