Solana CLI keypair basics

The Solana CLI stores the default keypair at:
    ~/.config/solana/id.json

This file is a JSON array of 64 bytes: the first 32 bytes are the private
key material, the last 32 bytes are the public key. The file is generated by:

    solana-keygen new --outfile ~/.config/solana/id.json

During generation, you are shown a seed phrase (BIP-39 mnemonic) and asked
to set an optional passphrase. Store the seed phrase securely — it can
recover the keypair if the file is lost.

View the public key (address) of the default keypair:
    solana address
    # or equivalently:
    solana-keygen pubkey ~/.config/solana/id.json
