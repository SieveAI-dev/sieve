Foundry cast wallet commands

Foundry's `cast wallet` subcommand manages keystores in a Foundry-native way:

    # Create a new encrypted keystore
    cast wallet new

    # Create in a specific directory
    cast wallet new --keystore-dir ./keystore

    # Show the address of a keystore
    cast wallet address --keystore ./keystore/UTC--*

    # Sign a message with a keystore (prompts for passphrase)
    cast wallet sign --keystore ./keystore/UTC--* "Hello"

    # Import an existing private key as a keystore
    cast wallet import mykey --interactive

The `--interactive` flag reads the private key from stdin without it
appearing in shell history. Keystores created by cast are fully compatible
with geth, MetaMask, and other EIP-55 compliant tools.
