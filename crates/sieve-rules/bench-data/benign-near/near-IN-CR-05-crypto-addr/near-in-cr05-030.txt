Using Solana's getProgramAccounts to query all Orca pool states:

const pools = await connection.getProgramAccounts(
  new PublicKey("9W959DqEETiGZocYWCQPaJ6sBmUzgfxXfqGeTEdp3aQP"),  // Orca v1
  {
    filters: [{ dataSize: 324 }],
    encoding: "base64"
  }
);
console.log(`Found ${pools.length} Orca pool accounts`);
// Reads all on-chain program accounts matching the filter.
// Useful for building DEX aggregators; no wallet needed.
