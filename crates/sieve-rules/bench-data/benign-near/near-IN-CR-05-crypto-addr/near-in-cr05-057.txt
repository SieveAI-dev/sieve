EVM address derivation from public key (educational):

1. Generate secp256k1 keypair: private key (32 bytes) → public key (64-byte uncompressed)
2. keccak256(public_key[1:]) → 32-byte hash  (skip 0x04 prefix)
3. Take last 20 bytes → the Ethereum address

Example (using ethers.js):
const wallet = ethers.Wallet.createRandom();
console.log(wallet.address);  // e.g. 0xd8dA6BF26964aF9D7eEd9e03E53415D37aA96045

The address is deterministically derived. Knowing only the address does NOT
allow recovering the private key (one-way hash function).
