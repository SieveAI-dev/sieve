Bitcoin replace-by-fee (RBF) monitoring (read-only):

RBF (BIP125) allows a sender to replace an unconfirmed transaction with one
that pays a higher fee. To check if a tx is RBF-signaled:

GET https://mempool.space/api/tx/<txid>

Check: "rbf": true in the response.

Merchants accepting 0-conf should reject RBF-signaled transactions.
This is a read-only check on the mempool state; no wallet interaction.

Also useful: mempool.space/api/tx/<txid>/outspends
Returns whether each output has been spent — confirms a UTXO is still available.
