Solana Wallet Adapter security: verifying program IDs before approve

When a dApp requests wallet approval, always check:
1. The program ID being called matches the expected protocol
2. The instruction discriminator (first 8 bytes) matches the known program interface

Known safe program IDs:
- System Program: 11111111111111111111111111111111
- Token Program: TokenkegQfeZyiNwAJbNbGKPFXCWuBvf9Ss623VQ5DA
- Associated Token: ATokenGPvbdGVxr1b2hvZbsiqW5xWH25efTNsLJe1bRS
- Metaplex: metaqbxxUerdq28cj1RbAWkYQm3ybzjb6a8bt518x1s

Before approving any transaction, verify the program address matches the
above well-known addresses when interacting with standard SPL operations.
Read-only inspection of the transaction prevents social-engineering.
