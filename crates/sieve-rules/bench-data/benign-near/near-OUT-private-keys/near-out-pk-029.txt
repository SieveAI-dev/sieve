SSH key type comparison (2024):

Ed25519: RECOMMENDED. 256-bit EdDSA. Small keys (~68 bytes public), fast, deterministic. No weak parameter choices possible. Supported since OpenSSH 6.5.

ECDSA P-256/P-384: Good choice. Based on NIST curves. Smaller than RSA. Slight implementation risk (random nonce reuse = catastrophic). Supported everywhere.

RSA-4096: Widely compatible (including legacy systems). Large keys. Use if Ed25519 not supported by target.

RSA-2048: Minimum acceptable. Being phased out.

DSA-1024: DEPRECATED. Do not use. Removed from OpenSSH 7.0 by default.

Never generate RSA < 2048. Ed25519 is preferred for all new keys. Generate with: ssh-keygen -t ed25519
