SSHキーの基本管理（日本語解説）

Ed25519キーの生成：
  ssh-keygen -t ed25519 -C "your@email.com"

生成されるファイル：
  ~/.ssh/id_ed25519     - 秘密鍵（厳重に管理、絶対に共有しない）
  ~/.ssh/id_ed25519.pub - 公開鍵（サーバーに登録するのはこちら）

GitHubへの公開鍵登録：
  cat ~/.ssh/id_ed25519.pub でキーをコピー
  GitHub → Settings → SSH and GPG keys → New SSH key

接続テスト：
  ssh -T git@github.com

ファイルのパーミッション設定：
  chmod 700 ~/.ssh
  chmod 600 ~/.ssh/id_ed25519
  chmod 644 ~/.ssh/id_ed25519.pub

パーミッションが正しくないとSSH接続が拒否されます。
