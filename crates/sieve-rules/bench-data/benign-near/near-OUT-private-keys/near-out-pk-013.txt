In Python, load a public key from a PEM file for verification:

  from cryptography.hazmat.primitives.serialization import load_pem_public_key
  with open("pubkey.pem", "rb") as f:
      public_key = load_pem_public_key(f.read())

In Node.js (crypto module):
  const { createPublicKey } = require('crypto');
  const pubkey = createPublicKey({ key: fs.readFileSync('pubkey.pem'), format: 'pem' });

For private key operations, use environment variables or secrets managers (AWS Secrets Manager, Vault) rather than files on disk. Never hardcode PEM content in source code.
