Keeping secrets OUT of git — tools and approaches:

.gitignore for key files (minimum baseline):
  *.pem
  *.key
  *.p12
  *.pfx
  id_rsa
  id_ed25519
  .env

git-crypt: transparent encryption for specific files:
  git-crypt init
  gpg --list-keys  # find your key ID
  git-crypt add-gpg-user YOUR_GPG_KEY_ID
  echo "secrets/*.key filter=git-crypt diff=git-crypt" >> .gitattributes

git-secrets: scan commits for secrets patterns:
  git secrets --install
  git secrets --register-aws
  git secrets --scan-history

Check if a key was accidentally committed:
  git log --all --full-history -- "*.pem" "*.key"
  git log -S "BEGIN PRIVATE KEY" --all
