System TLS trust store management — where root CA certificates (BEGIN CERTIFICATE blocks) live:

Linux (Debian/Ubuntu):
  /etc/ssl/certs/ca-certificates.crt    (bundle of trusted root certs)
  /usr/local/share/ca-certificates/     (drop custom .crt files here)
  sudo update-ca-certificates            (regenerate bundle)

Linux (RHEL/CentOS):
  /etc/pki/ca-trust/source/anchors/     (drop custom .pem files here)
  sudo update-ca-trust                   (regenerate)

macOS:
  System Keychain: /System/Library/Keychains/SystemRootCertificates.keychain
  Add via: sudo security add-trusted-cert -d -r trustRoot -k /Library/Keychains/System.keychain custom_ca.pem

Windows:
  certlm.msc → Trusted Root Certification Authorities → Certificates

Python override: REQUESTS_CA_BUNDLE=/path/to/custom_ca.pem python script.py
curl override: curl --cacert /path/to/custom_ca.pem https://internal.example.com
