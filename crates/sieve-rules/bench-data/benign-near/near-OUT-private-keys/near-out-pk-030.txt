SSH key fingerprints for verification without exposing key material:

Show fingerprint of a public key:
  ssh-keygen -l -f ~/.ssh/id_ed25519.pub
  # Output: 256 SHA256:abc123... user@host (ED25519)

Show fingerprint in old MD5 format (for legacy servers):
  ssh-keygen -l -E md5 -f ~/.ssh/id_ed25519.pub
  # Output: 256 MD5:aa:bb:cc:dd:... user@host (ED25519)

Visualize fingerprint as "randomart" ASCII art:
  ssh-keygen -lv -f ~/.ssh/id_ed25519.pub

Fingerprints are SHA-256 hashes of the public key — safe to compare publicly. GitHub shows fingerprints at github.com/settings/ssh — verify yours matches. Server host key fingerprint shown on first connection.
