SSH keygen options for RSA keys:

  ssh-keygen -t rsa -b 4096 -C "comment" -f ~/.ssh/my_rsa_key

Flags:
  -t rsa        key type (rsa, ecdsa, ed25519, dsa)
  -b 4096       key bits (only meaningful for RSA; Ed25519 ignores this)
  -C "comment"  comment appended to public key (typically email/hostname)
  -f path       output file path (default: ~/.ssh/id_<type>)
  -N ""         empty passphrase (skip interactive prompt)
  -q            quiet mode

The generated private key starts with "BEGIN OPENSSH PRIVATE KEY" header in modern OpenSSH format (since OpenSSH 7.8). Older PEM RSA format used "BEGIN RSA PRIVATE KEY".
