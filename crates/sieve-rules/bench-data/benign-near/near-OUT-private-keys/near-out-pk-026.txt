Adding SSH key to GitHub account:

1. Generate key if you don't have one:
   ssh-keygen -t ed25519 -C "github@example.com"

2. Copy public key (safe to share):
   cat ~/.ssh/id_ed25519.pub
   # Output example: ssh-ed25519 AAAAC3NzaC1lZDI1NTE5AAAAIExample... github@example.com

3. GitHub Settings → SSH and GPG keys → New SSH key → paste the public key

4. Test connection:
   ssh -T git@github.com
   # Expected: "Hi username! You've successfully authenticated..."

5. Clone with SSH:
   git clone git@github.com:username/repo.git
