Q: How do I check the expiration date of an SSL/TLS certificate from the command line?

A: Use openssl to parse the certificate PEM file directly:

  openssl x509 -in /path/to/cert.pem -noout -dates

Output:
  notBefore=Jan  1 00:00:00 2024 GMT
  notAfter=Jan  1 00:00:00 2025 GMT

For a remote server without downloading the cert:
  openssl s_client -connect example.com:443 -servername example.com \
    < /dev/null 2>/dev/null | openssl x509 -noout -dates

For bulk checking multiple certs in a directory:
  for f in /etc/nginx/ssl/*.pem; do
    echo -n "$f: "
    openssl x509 -noout -enddate -in "$f" 2>/dev/null || echo "not a cert"
  done

Automate alerts: use check_ssl_cert Nagios plugin or Prometheus ssl_certificate_expiry_seconds metric.
