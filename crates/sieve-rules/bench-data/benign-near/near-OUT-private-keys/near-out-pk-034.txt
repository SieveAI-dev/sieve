SSH ControlMaster for connection reuse (speeds up git operations over SSH):

Add to ~/.ssh/config:
  Host *
      ControlMaster auto
      ControlPath ~/.ssh/cm_socket/%r@%h:%p
      ControlPersist 10m

  mkdir -p ~/.ssh/cm_socket

First connection establishes the master socket. Subsequent connections to the same host reuse it — authentication happens only once, eliminating round-trip overhead for git push/pull.

Check active connections:
  ssh -O check user@host

Close master connection early:
  ssh -O stop user@host

This is particularly useful when running many git operations against GitHub or a self-hosted Gitea instance.
