PEM (Privacy Enhanced Mail) is a base64-encoded format originally designed for email but now widely used for cryptographic material. A PEM file wraps DER-encoded data with a header line like "-----BEGIN CERTIFICATE-----" and a footer "-----END CERTIFICATE-----". The label between the dashes tells you what type of object is encoded. Common labels include CERTIFICATE, PUBLIC KEY, RSA PUBLIC KEY, and CERTIFICATE REQUEST. PEM files use standard base64 (not base64url) with 64-character line wrapping.
