Debugging "unable to load private key" errors:

Error: "Enter PEM pass phrase:" + wrong password → wrong passphrase for encrypted key
Error: "bad password read" → passphrase incorrect or passphrase on stdin failed
Error: "no start line" → file is empty or not PEM format
Error: "PEM_read_bio_PrivateKey" → library failed to parse key

Diagnose with:
  openssl pkey -in key.pem -check -noout
  # If it asks for passphrase, key is encrypted
  # If it says "Key is valid", all good (without -noout it prints public part)

Check if key file is passphrase-protected:
  grep "ENCRYPTED" key.pem && echo "encrypted" || echo "not encrypted"

Test key + cert match:
  openssl verify -CAfile ca.crt server.crt
  diff <(openssl x509 -in server.crt -pubkey -noout) <(openssl pkey -in server.key -pubout)
