Generating an Ed25519 SSH key pair (recommended for new keys):

  ssh-keygen -t ed25519 -C "your.email@example.com"

This creates two files:
- ~/.ssh/id_ed25519       (private key - never share this file)
- ~/.ssh/id_ed25519.pub   (public key - safe to share, add to servers)

Set correct permissions:
  chmod 700 ~/.ssh
  chmod 600 ~/.ssh/id_ed25519
  chmod 644 ~/.ssh/id_ed25519.pub

To add a passphrase to an existing key:
  ssh-keygen -p -f ~/.ssh/id_ed25519
