PKCS#1 vs PKCS#8 private key format: When OpenSSL generates an RSA key with "openssl genrsa", the result is a PKCS#1 RSAPrivateKey. The PEM header reads "BEGIN RSA PRIVATE KEY" (without the dashes triggering a real key - this is just the label name in documentation). PKCS#8 wraps the same key in an outer PrivateKeyInfo structure and uses the label "BEGIN PRIVATE KEY". PKCS#8 is algorithm-agnostic and preferred for interoperability. Convert with: openssl pkcs8 -topk8 -nocrypt -in key_pkcs1.pem -out key_pkcs8.pem
