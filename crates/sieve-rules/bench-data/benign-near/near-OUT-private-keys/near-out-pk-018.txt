DER vs PEM: Two encodings of the same data. DER (Distinguished Encoding Rules) is binary — compact, unambiguous, not human-readable. PEM is DER encoded as base64 with BEGIN/END headers — human-readable ASCII, safe for copy-paste, email, config files. Most tools auto-detect format. To convert DER certificate to PEM: openssl x509 -inform DER -in cert.der -out cert.pem. Java typically uses DER/JKS keystores; web servers typically use PEM. File extensions: .pem, .crt, .cer, .key (all may be PEM); .der, .cer (may be DER binary). Check with: file cert.cer
