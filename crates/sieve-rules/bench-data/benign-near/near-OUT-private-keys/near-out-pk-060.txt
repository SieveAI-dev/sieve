Hardware Security Modules (HSM) and PKCS#11 interface:

PKCS#11 is a standard C API for interacting with cryptographic tokens (HSMs, smart cards, TPMs). Private keys stored in HSMs cannot be exported — signing operations happen inside the hardware.

OpenSSL with PKCS#11 engine (via libp11):
  openssl engine -t pkcs11
  openssl s_server -engine pkcs11 -keyform engine \
    -key "pkcs11:token=MyToken;object=server-key;type=private" \
    -cert server.crt

SoftHSM2 for testing (software emulation of HSM):
  apt install softhsm2
  softhsm2-util --init-token --slot 0 --label "TestToken"
  pkcs11-tool --module /usr/lib/softhsm/libsofthsm2.so --list-objects

In cloud environments: AWS CloudHSM, GCP Cloud HSM, Azure Dedicated HSM all provide PKCS#11 interfaces. Keys generated in HSMs are never exposed in plaintext.
