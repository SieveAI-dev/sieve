SSH tunneling and ProxyJump configuration:

Jump through bastion host:
  ssh -J user@bastion.example.com user@internal.host

In ~/.ssh/config:
  Host internal
      ProxyJump bastion
      IdentityFile ~/.ssh/id_ed25519

Forward local port to remote service (local port forwarding):
  ssh -L 5432:db.internal:5432 user@bastion
  # Then connect to localhost:5432

Forward remote port to local machine (reverse forwarding):
  ssh -R 8080:localhost:3000 user@server

SOCKS5 proxy through SSH:
  ssh -D 1080 user@server
  # Configure browser/curl to use SOCKS5 localhost:1080

Agent forwarding (use local keys on remote host):
  ssh -A user@server   # or ForwardAgent yes in config
  # Security note: ForwardAgent exposes keys to remote root; use only on trusted hosts
