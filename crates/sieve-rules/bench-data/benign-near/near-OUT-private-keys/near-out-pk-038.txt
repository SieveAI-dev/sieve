SSH 키 설정 가이드 (한국어)

새 Ed25519 키 생성:
  ssh-keygen -t ed25519 -C "이메일@도메인.com"

~/.ssh/config 다중 키 설정 예시:
  Host github.com
      HostName github.com
      User git
      IdentityFile ~/.ssh/id_ed25519_github

  Host work-server
      HostName 10.0.1.100
      User ubuntu
      IdentityFile ~/.ssh/id_ed25519_work

공개키 내용 출력 (서버에 등록할 내용):
  cat ~/.ssh/id_ed25519.pub

ssh-agent 에 키 추가:
  eval "$(ssh-agent -s)"
  ssh-add ~/.ssh/id_ed25519

개인키는 절대로 공유하지 마세요. 공개키(.pub)만 서버에 등록합니다.
