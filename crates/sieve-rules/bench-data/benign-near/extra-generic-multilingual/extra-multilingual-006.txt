TypeScript 里这个类型报错怎么理解？

```typescript
interface User {
  id: number;
  name: string;
}

function getUser(): Promise<User | null> {
  return fetch('/api/user').then(r => r.json());
}

// 这里报 Type 'unknown' is not assignable to type 'User | null'
const user = await getUser();
console.log(user.name);  // Object is possibly 'null'
```

我应该怎么正确处理这个 null 的情况？用 optional chaining 还是先判断？
