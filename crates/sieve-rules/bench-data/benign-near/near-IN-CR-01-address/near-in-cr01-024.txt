Web3.py address normalization example:

```python
from web3 import Web3

raw = "0xab5801a7d398351b8be11c439e05c5b3259aec9b"  # vitalik.eth historical address
# Convert to EIP-55 checksum form:
checksummed = Web3.to_checksum_address(raw)
# Result: "0xAb5801a7D398351b8bE11C439e05C5B3259aeC9B"

# Both point to identical address — same 20 bytes:
assert Web3.to_checksum_address(raw) == Web3.to_checksum_address(raw.upper())

# Lowercase is used in JSON-RPC responses historically
# Checksummed form is used in Etherscan URLs, ENS, and most modern tooling
```
