How to resolve ENS names in ethers.js v6:

```javascript
import { ethers } from "ethers";

const provider = new ethers.JsonRpcProvider("https://eth.llamarpc.com");

// Forward lookup: name → address
const addr = await provider.resolveName("vitalik.eth");
console.log(addr); // 0xd8dA6BF26964aF9D7eEd9e03E53415D37aA96045

// Reverse lookup: address → name
const name = await provider.lookupAddress("0xd8dA6BF26964aF9D7eEd9e03E53415D37aA96045");
console.log(name); // "vitalik.eth"

// ENS Registry on mainnet:
// 0x00000000000C2E074eC69A0dFb2997BA6C7d2e1e
```

Always do a round-trip check: resolve the name, then reverse-lookup the address
to confirm it matches. Phishing sites sometimes set up misleading ENS names.
