Cast (Foundry CLI) address checksum verification:

```bash
# Compute EIP-55 checksum of an address:
$ cast to-check-sum-address 0xfb6916095ca1df60bb79ce92ce3ea74c37c5d359
0xfB6916095ca1df60bB79Ce92cE3Ea74c37c5d359

# Verify two representations are the same address:
$ cast to-check-sum-address 0xFB6916095CA1DF60BB79CE92CE3EA74C37C5D359
0xfB6916095ca1df60bB79Ce92cE3Ea74c37c5d359
# Same output → same address

# This is the Ethereum Foundation donation address (public knowledge).
# Do not confuse case differences with different addresses.
```
