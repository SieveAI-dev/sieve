Proxy factory: 0x4e1DCf7AD4e460CfD30791CCC4F9c8a4f820ec67

Each Safe deployment is a minimal proxy pointing to this singleton. The proxy's
address varies per user (derived from salt = owner set + nonce). The singleton
itself lives at the same address on every supported network. See safe-deployments npm pkg.
