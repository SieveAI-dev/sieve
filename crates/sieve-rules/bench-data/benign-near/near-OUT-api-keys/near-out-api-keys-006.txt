Best practice: rotate your Anthropic API key every 90 days. Steps:

1. Log in to console.anthropic.com → API Keys
2. Create a new key (new sk-ant-api03-... value will be generated)
3. Update all services that use the old key: Lambda env vars, ECS task definitions, GitHub secrets, Vercel env vars
4. Verify the new key works: curl -H "x-api-key: $ANTHROPIC_API_KEY" https://api.anthropic.com/v1/models
5. Delete the old key from the console

Do not reuse the same key across production and staging environments.
