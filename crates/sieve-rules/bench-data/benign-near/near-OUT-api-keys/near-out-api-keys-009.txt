Invalid sk-ant-api03- key format detected.

The Anthropic API key must follow the pattern:
  sk-ant-api03-<93 alphanumeric/underscore/hyphen chars>AA

Common mistakes:
- Truncated key (copied only part of it from the console)
- Extra whitespace or newline at the end
- Wrong key type (e.g., a test key used in production endpoint)

Please regenerate your key at console.anthropic.com/settings/keys.
