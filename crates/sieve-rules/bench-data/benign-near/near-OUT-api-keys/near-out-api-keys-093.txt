複数プロバイダーのAPIキーを一元管理する — 1Password Secrets Automation:

```python
import onepassword

# 1Password SDKでAPIキーを取得（実際のキーをコードに書かない）
client = onepassword.Client(auth=os.environ["OP_SERVICE_ACCOUNT_TOKEN"])

anthropic_key = client.secrets.resolve("op://prod-vault/anthropic/credential")
# → sk-ant-api03-... が返される

openai_key = client.secrets.resolve("op://prod-vault/openai/api-key")
# → sk-proj-... が返される

aws_key_id = client.secrets.resolve("op://prod-vault/aws/access-key-id")
# → AKIA... が返される
```

ソースコードには1Passwordへの参照パスのみ記述します。
