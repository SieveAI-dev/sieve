GitHub Actions: using a PAT with the gh CLI

```yaml
steps:
  - name: Checkout
    uses: actions/checkout@v4
  - name: Create PR with gh CLI
    env:
      GH_TOKEN: ${{ secrets.GH_PAT }}  # ghp_... or fine-grained token
    run: |
      gh pr create --title "Automated PR" --body "Created by CI"
```

The built-in GITHUB_TOKEN has limited permissions. For cross-repo operations, add a PAT (ghp_...) as a repository secret named GH_PAT.
