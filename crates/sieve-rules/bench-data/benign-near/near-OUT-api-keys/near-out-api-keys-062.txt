`.env` example for projects using GitHub API:

```
# GitHub credentials
# Generate at: https://github.com/settings/tokens
# Classic format: ghp_XXXX... (36 chars after prefix)
# Fine-grained: github_pat_XXXX...
GITHUB_TOKEN=ghp_REPLACE_WITH_YOUR_TOKEN

# Optional: target org for API calls
GITHUB_ORG=my-organization
GITHUB_REPO=my-repo
```

Never commit this file. Ensure `.env` is in `.gitignore` before creating it.
