GitHub TOKEN 类型对照：

| 前缀 | 类型 | 使用场景 |
|------|------|---------|
| ghp_ | 个人访问令牌 (PAT) | 开发者个人 API 访问 |
| gho_ | OAuth App Token | 第三方应用授权 |
| ghu_ | GitHub App 用户令牌 | App 代表用户操作 |
| ghs_ | GitHub App 安装令牌 | App 代表安装本身操作 |
| ghr_ | GitHub App 刷新令牌 | 刷新用户令牌 |

所有类型均不应出现在源代码中，统一通过 GITHUB_TOKEN 环境变量注入。
