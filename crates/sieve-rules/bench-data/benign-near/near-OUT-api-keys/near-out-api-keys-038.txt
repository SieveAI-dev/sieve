Error: AWS credentials not found.

Attempted to load credentials from:
1. Environment variables (AWS_ACCESS_KEY_ID, AWS_SECRET_ACCESS_KEY)
2. AWS credentials file (~/.aws/credentials)
3. EC2 instance metadata service
4. ECS container credentials

Solution: Run `aws configure` and enter your AKIA... access key ID and secret access key.
For production, prefer IAM roles attached to the EC2/Lambda rather than embedding AKIA credentials.

Note: AKIAIOSFODNN7EXAMPLE is the placeholder in AWS docs — replace with your real key ID.
