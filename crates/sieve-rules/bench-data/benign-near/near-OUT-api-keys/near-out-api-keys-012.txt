## Anthropic API Key Security Checklist

- [ ] Key is stored in environment variable, not source code
- [ ] `.env` file is listed in `.gitignore`
- [ ] Key prefix `sk-ant-api03-` is never printed in logs
- [ ] Key is rotated every 90 days
- [ ] Separate keys for dev/staging/production workspaces
- [ ] API usage alerts configured in console.anthropic.com
- [ ] Key has minimum required permissions (no admin scope unless needed)
- [ ] Key is stored in a secrets manager (Vault, AWS SM, GCP Secret Manager)
