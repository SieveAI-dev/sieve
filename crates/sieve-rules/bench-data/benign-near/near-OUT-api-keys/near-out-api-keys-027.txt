## .env file template for OpenAI projects

```
# OpenAI Configuration
# Get your key at: https://platform.openai.com/api-keys
# Format: sk-proj-... (new) or sk-... (legacy)
OPENAI_API_KEY=sk-proj-REPLACE_WITH_YOUR_KEY

# Optional: Organization ID (found at platform.openai.com/account/org-settings)
OPENAI_ORG_ID=org-REPLACE_WITH_YOUR_ORG_ID

# Model selection
OPENAI_MODEL=gpt-4o
```

Add `.env` to `.gitignore` before committing this template (remove actual values first).
