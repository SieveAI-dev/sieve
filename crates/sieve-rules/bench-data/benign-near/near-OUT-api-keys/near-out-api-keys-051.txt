AWS CLI profile for multiple accounts:

```ini
# ~/.aws/credentials
[dev]
aws_access_key_id = AKIA_DEV_KEY_ID_REPLACE
aws_secret_access_key = dev-secret-replace-me

[prod]
aws_access_key_id = AKIA_PROD_KEY_ID_REPLACE
aws_secret_access_key = prod-secret-replace-me
```

Switch with: export AWS_PROFILE=prod
Or per-command: aws s3 ls --profile dev

Do not share the [prod] credentials file section with developers who only need [dev] access.
