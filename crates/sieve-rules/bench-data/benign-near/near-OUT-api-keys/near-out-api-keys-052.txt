GitHub Personal Access Tokens (PAT) come in two types:

Classic tokens: ghp_XXXX... (36 alphanumeric chars after ghp_)
Fine-grained tokens: github_pat_... (newer format with repository-scoped permissions)

Other prefix types:
- gho_: OAuth app tokens
- ghu_: GitHub Apps user-to-server tokens
- ghs_: GitHub Apps server-to-server tokens
- ghr_: refresh tokens for GitHub Apps

Store your token as GITHUB_TOKEN or GH_TOKEN environment variable — never in source code.
