Error: Bad credentials — GitHub API returned 401.

Possible causes:
1. The ghp_... or gho_... token has expired (classic PATs can expire)
2. The token was revoked at github.com/settings/tokens
3. Wrong token used (user token vs. app token vs. installation token)
4. Missing scope — repo operations need `repo` scope on classic PATs

Fix: Generate a new token at github.com/settings/tokens/new and set it as GH_TOKEN.

Check current token permissions: gh auth status
