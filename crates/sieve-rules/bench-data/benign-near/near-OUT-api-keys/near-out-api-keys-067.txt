GitHub API rate limiting with PAT tokens:

Unauthenticated: 60 requests/hour
Authenticated with ghp_...: 5000 requests/hour
GitHub App: 15000 requests/hour per installation

Check remaining rate limit:
```bash
curl -H "Authorization: Bearer $GITHUB_TOKEN" \
  https://api.github.com/rate_limit
```

If you see 403 with "rate limit exceeded", your ghp_... token may have insufficient scopes or the rate limit is exhausted. Upgrade to a GitHub App for higher limits.
