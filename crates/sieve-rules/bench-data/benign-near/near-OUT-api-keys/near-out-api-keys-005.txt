GitHub Actions workflow snippet for using the Anthropic API:

```yaml
jobs:
  run-claude:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - name: Call Claude API
        env:
          ANTHROPIC_API_KEY: ${{ secrets.ANTHROPIC_API_KEY }}
        run: python scripts/summarize.py
```

Store the key as a repository secret. The format when setting it in GitHub is the raw key starting with sk-ant-api03-...
