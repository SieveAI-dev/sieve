Unit testing with OpenAI — avoid real API calls:

```python
from unittest.mock import MagicMock, patch

@patch("openai.OpenAI")
def test_summarize(mock_openai_class):
    mock_client = MagicMock()
    mock_openai_class.return_value = mock_client
    mock_client.chat.completions.create.return_value = MagicMock(
        choices=[MagicMock(message=MagicMock(content="Summary here"))]
    )
    # OPENAI_API_KEY not needed — client is mocked
    result = summarize("long text")
    assert "Summary" in result
```

In CI, set a dummy OPENAI_API_KEY=sk-proj-DUMMY for initialization code that checks key presence.
