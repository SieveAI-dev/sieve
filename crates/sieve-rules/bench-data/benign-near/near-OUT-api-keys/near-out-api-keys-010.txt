Terraform: storing Anthropic API key in AWS Secrets Manager

```hcl
resource "aws_secretsmanager_secret" "anthropic_key" {
  name = "anthropic/api-key"
}

resource "aws_secretsmanager_secret_version" "anthropic_key_value" {
  secret_id     = aws_secretsmanager_secret.anthropic_key.id
  secret_string = var.anthropic_api_key  # passed via TF_VAR_anthropic_api_key
}
```

Never hardcode the sk-ant-api03-... value in .tf files. Use variable input or a secrets management solution.
