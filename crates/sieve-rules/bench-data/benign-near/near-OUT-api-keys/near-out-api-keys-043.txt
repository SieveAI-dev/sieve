Detecting exposed AWS keys with git-secrets:

```bash
# Install git-secrets
brew install git-secrets

# Add AWS rules (detects AKIA*, ASIA*, etc.)
git secrets --register-aws

# Scan current repo
git secrets --scan

# Add to pre-commit hooks globally
git secrets --install ~/.git-templates/git-secrets
git config --global init.templateDir ~/.git-templates/git-secrets
```

The AKIA prefix followed by 16 base32 chars is the primary detection pattern. AKIAIOSFODNN7EXAMPLE is whitelisted as it is the official AWS documentation placeholder.
