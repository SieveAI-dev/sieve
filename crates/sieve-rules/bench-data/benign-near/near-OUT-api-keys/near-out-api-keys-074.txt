Google Maps API キーの管理（AIzaSy-...）:

APIキーは console.cloud.google.com で生成します。形式は AIzaSy- で始まる39文字です。

セキュリティのベストプラクティス：
1. HTTPリファラー制限を設定（本番ドメインのみ許可）
2. APIを特定のサービスに制限（例：Maps JavaScript APIのみ）
3. ソースコードに直接記述しない：
   ```js
   const apiKey = process.env.GOOGLE_MAPS_API_KEY; // AIzaSy-...
   ```
4. 90日ごとにローテーション
5. 漏洩した場合は console.cloud.google.com で即座に削除
