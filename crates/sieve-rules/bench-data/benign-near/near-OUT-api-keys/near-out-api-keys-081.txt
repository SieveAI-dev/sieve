Detecting leaked GCP API keys (AIza prefix) with trufflehog:

```bash
# Scan git history for GCP API keys
trufflehog git https://github.com/org/repo.git --only-verified

# Local scan
trufflehog filesystem ./src/

# trufflehog detects AIzaSy-... pattern and validates against GCP
# Unverified: pattern matched but key may be expired/revoked
# Verified: key is active and callable — rotate immediately!
```

The AIzaSy- prefix followed by 35 chars is the GCP API key signature. Trufflehog will attempt to verify it against the GCP API.
