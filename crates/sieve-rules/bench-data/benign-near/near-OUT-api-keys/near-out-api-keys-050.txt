AWS Bedrock also uses AKIA/ASIA credentials — the access key ID format is the same as standard AWS keys. However, Bedrock keys (ABIA prefix) are scoped to Bedrock inference profiles.

To call Bedrock from Python:
```python
import boto3
client = boto3.client("bedrock-runtime", region_name="us-east-1")
# Credentials from environment: AWS_ACCESS_KEY_ID (AKIA or ASIA) + AWS_SECRET_ACCESS_KEY
response = client.invoke_model(modelId="anthropic.claude-3-5-sonnet-20241022-v2:0", ...)
```

Set credentials via `aws configure` or IAM role — never embed AKIA values in code.
