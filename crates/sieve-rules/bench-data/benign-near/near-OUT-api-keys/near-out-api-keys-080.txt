GCP API 키 (AIzaSy-...) 모범 사례 — Next.js 프로젝트:

```js
// next.config.js
module.exports = {
  env: {
    // 서버 사이드 전용 (클라이언트에 노출 안 됨)
    GOOGLE_MAPS_API_KEY: process.env.GOOGLE_MAPS_API_KEY,
  },
  publicRuntimeConfig: {
    // 클라이언트 노출이 필요한 경우 — 반드시 도메인 제한 설정 필요
    NEXT_PUBLIC_MAPS_KEY: process.env.NEXT_PUBLIC_MAPS_KEY,
  }
}
```

NEXT_PUBLIC_ 접두사가 붙은 AIzaSy-... 키는 브라우저에 노출됩니다. 반드시 GCP 콘솔에서 HTTP 리퍼러 제한을 설정하세요.
