CI/CD pipeline secret injection patterns — multi-provider:

```yaml
# .github/workflows/deploy.yml
env:
  # Each key fetched from GitHub Secrets — never in YAML values
  ANTHROPIC_API_KEY: ${{ secrets.ANTHROPIC_API_KEY }}   # sk-ant-api03-...
  OPENAI_API_KEY: ${{ secrets.OPENAI_API_KEY }}         # sk-proj-...
  AWS_ACCESS_KEY_ID: ${{ secrets.AWS_ACCESS_KEY_ID }}   # AKIA...
  AWS_SECRET_ACCESS_KEY: ${{ secrets.AWS_SECRET_ACCESS_KEY }}
  GITHUB_TOKEN: ${{ secrets.GH_PAT }}                   # ghp_...
  GOOGLE_API_KEY: ${{ secrets.GOOGLE_API_KEY }}         # AIzaSy-...
```

GitHub Actions masks these values in log output. Do not `echo` them explicitly — they may appear if passed through shell variable expansion.
