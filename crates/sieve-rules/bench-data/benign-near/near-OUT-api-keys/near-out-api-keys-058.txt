GitHub API authentication in Python with a PAT:

```python
import os
from github import Github

# PyGitHub reads from environment variable
g = Github(os.environ["GITHUB_TOKEN"])  # expects ghp_... or gho_... token

# List repos
for repo in g.get_user().get_repos():
    print(repo.name)
```

The GITHUB_TOKEN environment variable should contain your ghp_... PAT. For GitHub Actions, the built-in secrets.GITHUB_TOKEN is a ghs_ scoped token.
