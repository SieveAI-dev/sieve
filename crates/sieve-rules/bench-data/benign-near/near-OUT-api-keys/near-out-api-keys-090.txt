Secrets management with Doppler — multi-provider key setup:

```bash
# Install and authenticate
brew install dopplerhq/cli/doppler
doppler login

# Set up project
doppler setup --project my-app --config prod

# Add secrets (values stored in Doppler, not local)
doppler secrets set ANTHROPIC_API_KEY=sk-ant-api03-YOUR_KEY
doppler secrets set OPENAI_API_KEY=sk-proj-YOUR_KEY
doppler secrets set AWS_ACCESS_KEY_ID=AKIA_YOUR_KEY_ID
doppler secrets set GITHUB_TOKEN=ghp_YOUR_TOKEN
doppler secrets set GOOGLE_API_KEY=AIzaSy-YOUR_GCP_KEY

# Run app with secrets injected
doppler run -- python main.py
```
