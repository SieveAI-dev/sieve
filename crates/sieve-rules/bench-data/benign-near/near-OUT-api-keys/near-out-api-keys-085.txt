GCP APIキー（AIzaSy-...）のセキュリティチェックリスト:

- [ ] アプリケーション制限を設定済み（HTTPリファラー or IPアドレス）
- [ ] API制限を設定済み（必要なAPIのみ）
- [ ] ソースコードにキーを直接記述していない
- [ ] .env ファイルを .gitignore に追加済み
- [ ] Cloud Monitoring でAPIの使用量アラートを設定済み
- [ ] キーのローテーションを90日以内に設定済み
- [ ] Google Cloud の予算アラートを設定済み（課金超過防止）
- [ ] 本番環境と開発環境で別々のキーを使用している
