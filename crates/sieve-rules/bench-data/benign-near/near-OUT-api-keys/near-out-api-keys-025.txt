Using OpenAI API with LangChain — environment setup:

```python
import os
from langchain_openai import ChatOpenAI

# LangChain reads OPENAI_API_KEY from environment automatically
llm = ChatOpenAI(model="gpt-4o")

# Or pass explicitly (still from env, not hardcoded):
llm = ChatOpenAI(
    model="gpt-4o",
    openai_api_key=os.getenv("OPENAI_API_KEY"),
)
```

The key should start with sk-proj- for new project-scoped keys, or sk- for legacy personal keys.
