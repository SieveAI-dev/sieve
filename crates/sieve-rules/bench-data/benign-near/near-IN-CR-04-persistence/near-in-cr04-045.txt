설정 셸 rc 파일 구성 가이드 (bash/zsh, 한국어)

셸 시작 파일의 역할 구분:

~/.bash_profile (또는 ~/.zprofile)
  - 로그인 셸에서 한 번만 읽힘
  - PATH, 환경 변수 설정에 적합

~/.bashrc (또는 ~/.zshrc)
  - 모든 대화형 셸에서 읽힘
  - 별칭(alias), 함수, 프롬프트 설정에 적합

현재 설정 내용 확인 (읽기 전용):
  cat ~/.bashrc
  cat ~/.zshrc

환경 변수 목록 확인:
  env | sort

PATH 확인:
  echo $PATH | tr ':' '\n'

셸 재시작 없이 변경 사항 적용:
  source ~/.zshrc   # zsh
  source ~/.bashrc  # bash

보통 .zshrc에 넣는 것들:
  - alias ll='ls -la'
  - 버전 관리자 초기화 (mise, asdf, nvm 등)
  - 자동완성 스크립트 로드
  - 프롬프트 테마 설정
