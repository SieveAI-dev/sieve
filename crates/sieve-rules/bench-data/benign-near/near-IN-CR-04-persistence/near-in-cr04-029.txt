Configuración del shell .zshrc para desarrolladores (español)

El archivo ~/.zshrc se ejecuta en cada sesión interactiva de Zsh. A continuación se muestran personalizaciones típicas para un entorno de desarrollo:

# Historial mejorado
HISTSIZE=50000
SAVEHIST=50000
setopt HIST_IGNORE_DUPS
setopt SHARE_HISTORY

# Directorio de binarios locales
export PATH="$HOME/.local/bin:$PATH"

# Alias habituales
alias g='git'
alias k='kubectl'
alias tf='terraform'

# Activar autocompletado mejorado
autoload -Uz compinit && compinit

# Integración con direnv
eval "$(direnv hook zsh)"

Para ver el contenido actual de tu .zshrc sin modificarlo:
  cat ~/.zshrc

Para recargarlo en la sesión actual:
  source ~/.zshrc

Nota: source es una operación de solo lectura sobre el proceso de shell actual; no escribe ningún archivo.
