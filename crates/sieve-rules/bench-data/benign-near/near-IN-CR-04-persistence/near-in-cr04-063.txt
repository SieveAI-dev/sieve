Fish shell の universal variables の仕組み（日本語）

Fish には他のシェルにはない「universal variables（ユニバーサル変数）」という仕組みがあります。

**通常の変数との違い**：
  set FOO "bar"         # ローカル（現在のセッションのみ）
  set -x FOO "bar"      # エクスポート（子プロセスに継承）
  set -U FOO "bar"      # ユニバーサル（全セッション・再起動後も保持）

**ユニバーサル変数の特徴**：
- ~/.config/fish/fish_variables に自動保存される
- 変更がリアルタイムで他の fish セッションに反映される
- 明示的に source する必要がない

**現在のユニバーサル変数を確認する**：
  set --universal   # または set -U の一覧

**注意点**：
- ユニバーサル変数はマシン固有の設定（API キーなど）に使いがち
- チームで共有するツール設定は config.fish に書いてGit管理する方が良い
- 機密情報はユニバーサル変数に入れず、1Password CLI などのシークレット管理ツールを使う

fish_variables ファイルはバイナリ形式に近いため、直接編集は推奨されません。set コマンド経由で操作してください。
