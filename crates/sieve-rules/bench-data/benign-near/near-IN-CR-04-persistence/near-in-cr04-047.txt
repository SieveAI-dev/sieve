fish_config: the browser-based Fish configuration UI.

Fish shell ships with a built-in web-based configuration tool. Run:
  fish_config

This opens a local web UI (typically at http://localhost:8000/) that lets you:
- Browse and preview available color themes
- Inspect and modify prompt appearance
- View defined functions, aliases, abbreviations, and variables

Changes made through fish_config are saved automatically as universal variables or function files. No manual file editing required.

fish_config is the recommended starting point for new Fish users who want to customize their shell without learning the config.fish syntax first.

Sub-commands available from the CLI:
  fish_config theme list       # list installed themes
  fish_config theme show       # print theme colors to terminal
  fish_config prompt list      # list available prompt styles
  fish_config prompt show      # preview prompt styles
  fish_config prompt choose    # interactively select a prompt

None of these commands write to config.fish directly; they operate through Fish's universal variable and function systems.
