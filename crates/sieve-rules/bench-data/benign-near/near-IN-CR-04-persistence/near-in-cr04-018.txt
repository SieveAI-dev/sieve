## Setting up oh-my-zsh themes and plugins

oh-my-zsh is a community-maintained framework for managing your Zsh configuration. After installation (via the official installer script which you run once), your ~/.zshrc will contain an `source "$ZSH/oh-my-zsh.sh"` line.

Key configuration points in ~/.zshrc (these are the fields you edit, not new lines you append):

```zsh
# Change theme (edit, not add)
ZSH_THEME="robbyrussell"

# Enable plugins (edit the plugins array)
plugins=(git docker kubectl fzf)
```

After editing those values, reload with `exec zsh`.

Popular lightweight themes: `agnoster`, `powerlevel10k` (requires a Nerd Font), `minimal`.

Plugin highlights:
- `git`: loads dozens of git aliases (view with `alias | grep "^g"`)
- `fzf`: integrates fuzzy finder for history and file search
- `kubectl`: adds tab completion for kubectl commands

You never need to manually append lines to .zshrc for oh-my-zsh feature activation — all standard features are controlled via the two config arrays above.
