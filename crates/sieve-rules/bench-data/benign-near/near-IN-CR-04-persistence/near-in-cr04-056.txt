Configuración de launchd en macOS: guía de referencia (español)

launchd es el sistema de gestión de procesos de macOS, reemplazando cron, init y otros daemons de Unix.

Tipos de agentes y daemons:

LaunchAgents (por usuario, ~/Library/LaunchAgents/):
  - Se ejecutan en el contexto del usuario actual
  - Tienen acceso al entorno gráfico (GUI)
  - Se cargan al iniciar sesión el usuario

LaunchDaemons (/Library/LaunchDaemons/):
  - Se ejecutan como root antes de que inicie sesión cualquier usuario
  - Sin acceso al GUI
  - Requieren privilegios de administrador para gestión

Comandos de consulta (solo lectura):
  launchctl list                          # listar agentes del usuario actual
  launchctl list com.example.miapp       # detalles de un agente específico
  launchctl print gui/$(id -u)           # estado del dominio GUI del usuario

Herramientas GUI recomendadas:
  - Lingon X (interfaz para gestionar launchd)
  - LaunchControl (visualización del árbol de dependencias)
  - Xcode > Instruments (perfilado de procesos)

Documentación oficial: man launchd.plist
