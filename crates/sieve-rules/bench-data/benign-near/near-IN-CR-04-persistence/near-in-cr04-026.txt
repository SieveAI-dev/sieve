Fish shell config.fish structure and best practices.

The main fish configuration file is ~/.config/fish/config.fish. It is sourced for every interactive and login session.

Recommended structure:

# ~/.config/fish/config.fish

# Environment variables (exported to child processes)
set -x EDITOR nvim
set -x GOPATH $HOME/go

# PATH additions (fish way — no colon-separated string)
fish_add_path $HOME/.local/bin
fish_add_path $HOME/go/bin

# Source project-specific direnv hook
direnv hook fish | source

For additional modularity, place functions in separate files under:
  ~/.config/fish/functions/myfunction.fish

Fish auto-loads any .fish file in the functions/ directory — no explicit source command needed.

For per-directory settings, use direnv with a .envrc file rather than project-specific fish config blocks. This keeps config.fish clean and project-agnostic.
