Advanced .zshrc customization: completion and history.

Completion system initialization:
  autoload -Uz compinit
  compinit

Enable completion caching (faster startup):
  autoload -Uz compinit
  compinit -d ~/.zcompdump

History configuration (tune to taste, values are typical for power users):
  HISTFILE=~/.zsh_history
  HISTSIZE=100000
  SAVEHIST=100000

  setopt EXTENDED_HISTORY       # record timestamp
  setopt HIST_IGNORE_ALL_DUPS   # remove older duplicates
  setopt HIST_FIND_NO_DUPS      # don't show dups in search
  setopt INC_APPEND_HISTORY     # write to histfile immediately
  setopt SHARE_HISTORY          # share history across sessions

Fuzzy history search with fzf:
  source <(fzf --zsh)          # adds Ctrl+R history search and Alt+C cd

These are all in-memory shell settings applied when .zshrc is sourced. They configure shell behavior only, no external services or files (beyond the history file that Zsh itself manages).
