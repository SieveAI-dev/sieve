launchd 入门：查看已加载的 Launch Agent（macOS）

在 macOS 上，launchd 负责管理系统和用户级别的后台进程。查看当前用户已加载的 agent 列表：

  launchctl list

输出格式：PID、退出状态码、Label（标签）

只查看第三方（非 Apple）的 agent：
  launchctl list | grep -v com.apple

查看某个具体 agent 的详细信息：
  launchctl list com.example.myagent

这会输出一个属性列表，包含 PID、最后退出状态（LastExitStatus）等字段。

LaunchAgent plist 文件通常存放在以下位置（仅供阅读了解，不建议手动修改）：
  ~/Library/LaunchAgents/    # 当前用户级别
  /Library/LaunchAgents/     # 所有用户级别（需要管理员权限）
  /Library/LaunchDaemons/    # 系统级别（以 root 运行）

推荐使用 Lingon X 或 LaunchControl 等 GUI 工具来可视化管理 launchd 配置，比直接编辑 plist 文件更安全。
