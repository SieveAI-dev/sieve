Cron expression syntax: five fields separated by spaces.
  * * * * *
  | | | | |
  | | | | +-- Day of week (0-7, Sunday=0 or 7)
  | | | +---- Month (1-12)
  | | +------ Day of month (1-31)
  | +-------- Hour (0-23)
  +---------- Minute (0-59)

Example: "30 8 * * 1-5" means every weekday at 08:30.
Special strings: @reboot, @daily, @hourly, @weekly, @monthly.

To view your current crontab entries: crontab -l
To view root's crontab: sudo crontab -l -u root
