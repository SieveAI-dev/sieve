# dev.to-style Article: JWT in 5 minutes

TL;DR: A JWT is a compact, URL-safe way to represent claims between two parties.

```javascript
// Sign a token
const token = jwt.sign(
  { userId: 42, role: 'user' },
  process.env.JWT_SECRET,
  { algorithm: 'HS256', expiresIn: '1h' }
);

// Verify a token
try {
  const payload = jwt.verify(token, process.env.JWT_SECRET);
  console.log(payload.userId); // 42
} catch (err) {
  // TokenExpiredError, JsonWebTokenError, etc.
  res.status(401).json({ error: 'Unauthorized' });
}
```

The token looks like: `eyJhbGciOiJIUzI1NiJ9.eyJzdWIiOiI0MiJ9.EXAMPLE_SIG_HERE`
Three base64url segments joined by dots. Don't try to decode this demo token—it's not signed.
