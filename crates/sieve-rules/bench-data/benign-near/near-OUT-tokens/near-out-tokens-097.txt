# Discord Bot CI/CD with GitHub Actions

```yaml
name: Test Bot
on: [push, pull_request]
jobs:
  test:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-python@v5
        with:
          python-version: '3.12'
      - run: pip install discord.py pytest
      - run: pytest tests/
        env:
          DISCORD_BOT_TOKEN: ${{ secrets.DISCORD_BOT_TOKEN }}
```

For unit tests, mock the client:
```python
from unittest.mock import AsyncMock, MagicMock

async def test_ping_command():
    interaction = MagicMock()
    interaction.response.send_message = AsyncMock()
    await ping(interaction)
    interaction.response.send_message.assert_called_once_with("Pong!")
```

Integration tests use DISCORD_BOT_TOKEN. Unit tests mock discord.py.
