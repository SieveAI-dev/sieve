# Slack API Rate Limits

Slack enforces per-method rate limiting (Tier 1-4):

Tier 1 (1 request/min): admin.*, conversations.create
Tier 2 (20 requests/min): channels.list, users.list
Tier 3 (50 requests/min): chat.postMessage, files.upload
Tier 4 (100 requests/min): auth.test, conversations.history

When rate limited, Slack returns HTTP 429 with Retry-After header.

Implement backoff:
```python
def api_call_with_retry(client, method, **kwargs):
    for attempt in range(5):
        response = getattr(client, method)(**kwargs)
        if response.get("error") == "ratelimited":
            import time
            time.sleep(2 ** attempt)
            continue
        return response
```

Token (xoxb-...) is unrelated to rate limits—limits are per application (client_id).
