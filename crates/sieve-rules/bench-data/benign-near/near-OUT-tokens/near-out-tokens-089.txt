Discord Bot Error: "LoginFailure: Improper token has been passed"

This error occurs when `client.run()` receives an invalid token.

Common causes:
1. DISCORD_BOT_TOKEN not set: os.environ['DISCORD_BOT_TOKEN'] raises KeyError
2. Wrong token: Using client secret instead of bot token
3. Stale token: Token regenerated after a security incident
4. Whitespace: Leading/trailing spaces in the token string

Debugging:
```python
token = os.environ.get('DISCORD_BOT_TOKEN', '')
print(f"Token length: {len(token)}")
print(f"Token prefix (first 10 chars): {token[:10]}")
# Real tokens are 70-80 characters long
# Do NOT print the full token
```

Fix: Regenerate token in discord.com/developers/applications → Bot → Reset Token.
Update all environments with the new token.
