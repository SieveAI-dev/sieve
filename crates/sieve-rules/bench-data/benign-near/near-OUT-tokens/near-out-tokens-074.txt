Slack API를 이용한 채널 목록 조회 (한국어)

conversations.list API 사용 예시:

```python
from slack_sdk import WebClient
import os

client = WebClient(token=os.environ["SLACK_BOT_TOKEN"])  # xoxb-...

result = client.conversations_list(
    types="public_channel",
    exclude_archived=True,
    limit=200
)

channels = result["channels"]
for channel in channels:
    print(f"채널: #{channel['name']}, ID: {channel['id']}")

# 페이지네이션 처리
cursor = result.get("response_metadata", {}).get("next_cursor")
while cursor:
    result = client.conversations_list(cursor=cursor, types="public_channel")
    channels.extend(result["channels"])
    cursor = result.get("response_metadata", {}).get("next_cursor")
```

필요한 스코프: channels:read
