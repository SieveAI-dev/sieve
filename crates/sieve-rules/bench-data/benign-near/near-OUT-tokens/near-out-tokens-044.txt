## Stripe Checkout: Hosted Payment Page

Stripe Checkout provides a pre-built payment UI. Minimal code required:

```python
session = stripe.checkout.Session.create(
    payment_method_types=['card'],
    line_items=[{
        'price_data': {
            'currency': 'usd',
            'product_data': {'name': 'Pro Plan'},
            'unit_amount': 2900,  # $29.00
        },
        'quantity': 1,
    }],
    mode='payment',
    success_url='https://example.com/success?session_id={CHECKOUT_SESSION_ID}',
    cancel_url='https://example.com/cancel',
)
redirect(session.url)
```

In test mode (sk_test_*), you'll be redirected to Stripe's test checkout page.
Payment succeeds with card 4242 4242 4242 4242, any future date, any CVC.
