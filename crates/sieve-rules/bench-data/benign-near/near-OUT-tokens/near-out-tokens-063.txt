## Slack Slash Command Request Verification

```python
import hmac
import hashlib
import time

def verify_slack_request(signing_secret: str, body: bytes, timestamp: str, signature: str) -> bool:
    # Prevent replay attacks
    if abs(time.time() - float(timestamp)) > 300:
        return False
    
    sig_basestring = f"v0:{timestamp}:{body.decode('utf-8')}"
    computed = "v0=" + hmac.new(
        signing_secret.encode(),
        sig_basestring.encode(),
        hashlib.sha256
    ).hexdigest()
    
    return hmac.compare_digest(computed, signature)
```

Note: The signing secret is separate from the bot token (xoxb-...).
The signing secret is used to verify incoming requests from Slack.
Found in App Settings → Basic Information → App Credentials → Signing Secret.
