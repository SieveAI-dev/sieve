# RFC 7519 Quick Reference

JSON Web Token (JWT) is defined in RFC 7519 (May 2015).
Related standards:
  RFC 7515 - JSON Web Signature (JWS)
  RFC 7516 - JSON Web Encryption (JWE)
  RFC 7517 - JSON Web Key (JWK)
  RFC 7518 - JSON Web Algorithms (JWA)
  RFC 7797 - JWS Unencoded Payload Option

A JWT is a JWS or JWE with a JSON payload.
A signed JWT (JWS compact serialization): BASE64URL(header).BASE64URL(payload).BASE64URL(signature)
An encrypted JWT (JWE compact serialization): 5 parts separated by dots.
