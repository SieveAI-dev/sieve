## Slack Bolt (Python) Quick Start

```python
from slack_bolt import App
import os

app = App(token=os.environ["SLACK_BOT_TOKEN"])  # xoxb-... from env

@app.message("hello")
def message_hello(message, say):
    say(f"Hey there <@{message['user']}>!")

@app.command("/greet")
def handle_greet(ack, respond, command):
    ack()
    respond(f"Hello, {command['text']}!")

if __name__ == "__main__":
    app.start(port=int(os.environ.get("PORT", 3000)))
```

Environment setup:
```bash
export SLACK_BOT_TOKEN="xoxb-EXAMPLE-REPLACE-ME-WITH-REAL-TOKEN"
export SLACK_SIGNING_SECRET="your-signing-secret-here"
```

The token above is a placeholder. Get your real token from api.slack.com/apps.
