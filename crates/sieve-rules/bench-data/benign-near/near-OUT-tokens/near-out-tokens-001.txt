# Understanding JWT Structure

A JSON Web Token consists of three Base64URL-encoded parts separated by dots:
  header.payload.signature

Example header decoded: {"alg":"HS256","typ":"JWT"}
Example payload decoded: {"sub":"1234567890","name":"Alice","iat":1516239022}

The signature is computed as: HMACSHA256(base64url(header) + "." + base64url(payload), secret)

For learning purposes, use jwt.io to decode tokens. Never paste real tokens into public sites.
Always verify the signature server-side before trusting any claim.
