Slack Token Troubleshooting: "missing_scope"

Error: {"ok":false,"error":"missing_scope","needed":"chat:write","provided":"..."}

This means the bot token (xoxb-...) lacks a required scope.

Fix:
1. Go to api.slack.com/apps → Your App → Features → OAuth & Permissions
2. Add the missing scope under "Bot Token Scopes"
3. Reinstall the app (required after scope changes)
4. New xoxb-... token now includes the added scope

Note: Reinstallation invalidates the old token and issues a new one.
Update your SLACK_BOT_TOKEN environment variable with the new token.

Scope changes are cumulative—adding scopes is safe.
Removing scopes requires reinstallation and revokes capabilities.
