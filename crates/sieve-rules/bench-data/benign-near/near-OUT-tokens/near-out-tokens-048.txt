# stripe CLI completion for zsh

```zsh
# Add to ~/.zshrc
if command -v stripe &>/dev/null; then
  source <(stripe completion --shell zsh)
fi
```

Common stripe CLI commands:
  stripe login                  # Authenticate (stores test key locally)
  stripe config --list          # Show current configuration
  stripe listen                 # Forward webhooks to localhost
  stripe trigger <event>        # Send a test event
  stripe logs tail              # Stream API request logs
  stripe customers list         # List test customers
  stripe payment_intents list   # List test payment intents

The CLI automatically uses your test key after `stripe login`.
To use a specific key: stripe --api-key sk_test_XXXX <command>
