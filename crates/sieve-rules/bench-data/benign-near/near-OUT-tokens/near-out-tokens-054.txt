Slack Token Rotation and Revocation

If a Slack token (xoxb-...) is exposed:

1. Immediately: Revoke the token in the Slack App Dashboard
   App Settings → Features → OAuth & Permissions → Revoke Token

2. Audit: Check Slack audit logs for unauthorized activity
   (Available on Enterprise Grid plans)

3. Rotate: Generate a new token by reinstalling the app
   Revoking and reinstalling issues a fresh xoxb-... token

4. Prevent recurrence:
   - Never log the full token value
   - Use secret scanning (GitHub has built-in detection for xoxb-*)
   - Store tokens in a secrets manager
