Discord Bot Token Regeneration

If your token is compromised or accidentally exposed:

1. Go to discord.com/developers/applications
2. Select your application
3. Navigate to Bot
4. Click "Regenerate Token" → Confirm

After regeneration:
- Old token is immediately invalidated
- Running bots using old token disconnect and cannot reconnect
- Update DISCORD_BOT_TOKEN in all environments
- Restart your bot with the new token

Automated detection: Discord's partner GitHub secret scanning automatically
invalidates tokens found in public repositories.

Best practice: Use a secrets manager and deploy with environment injection.
