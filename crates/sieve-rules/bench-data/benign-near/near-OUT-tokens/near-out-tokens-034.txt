Stripe 결제 연동 가이드 (한국어)

Stripe API 키 종류:
- sk_test_*: 테스트용 시크릿 키 (개발 및 CI에서 사용)
- pk_test_*: 테스트용 공개 키 (프론트엔드에서 사용 가능)
- sk_live_*: 실제 결제용 시크릿 키 (절대 노출 금지)
- pk_live_*: 실제 결제용 공개 키

기본 설정 예시:
```python
import stripe
import os

stripe.api_key = os.environ.get("STRIPE_SECRET_KEY")
# 개발환경: sk_test_로 시작하는 키 사용
# 운영환경: 환경변수에서 sk_live_로 시작하는 키 로드
```

보안 주의사항:
- .env 파일을 .gitignore에 추가하세요
- sk_live_ 키는 절대 소스코드에 포함시키지 마세요
- 정기적으로 키를 갱신하세요
