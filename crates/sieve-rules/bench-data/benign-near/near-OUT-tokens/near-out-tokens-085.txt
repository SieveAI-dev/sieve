Discord 봇 개발 가이드 (한국어)

Discord 봇 기본 설정:

1. 개발자 포털 (discord.com/developers/applications)
   - New Application 클릭 → Bot 추가
   - DISCORD_BOT_TOKEN 복사 후 안전하게 보관

2. 파이썬 봇 기본 코드:
```python
import discord, os

client = discord.Client(intents=discord.Intents.default())

@client.event
async def on_ready():
    print(f'{client.user}로 로그인했습니다')

@client.event
async def on_message(message):
    if message.content == '!안녕':
        await message.channel.send('안녕하세요!')

client.run(os.environ['DISCORD_BOT_TOKEN'])
```
