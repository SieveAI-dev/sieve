# Slack Bot Development Tutorial: Building a /weather command

```javascript
const { App } = require('@slack/bolt');

const app = new App({
  token: process.env.SLACK_BOT_TOKEN,    // xoxb-... stored in env
  signingSecret: process.env.SLACK_SIGNING_SECRET
});

app.command('/weather', async ({ command, ack, respond }) => {
  await ack();
  const city = command.text;
  const weather = await fetchWeather(city);  // Your weather API call
  await respond(`Weather in ${city}: ${weather.description}, ${weather.temp}°C`);
});

(async () => {
  await app.start(process.env.PORT || 3000);
  console.log('Bolt app is running!');
})();
```

Install: npm install @slack/bolt
Run: SLACK_BOT_TOKEN=xoxb-EXAMPLE SLACK_SIGNING_SECRET=secret node app.js
