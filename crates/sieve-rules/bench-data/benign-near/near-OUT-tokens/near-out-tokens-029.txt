# Stripe Node.js Quick Start (Test Mode)

```typescript
import Stripe from 'stripe';

const stripe = new Stripe(process.env.STRIPE_SECRET_KEY!, {
  apiVersion: '2024-06-20',
});

// Create a PaymentIntent (test mode)
const paymentIntent = await stripe.paymentIntents.create({
  amount: 2000,  // $20.00 in cents
  currency: 'usd',
  automatic_payment_methods: { enabled: true },
});

console.log(paymentIntent.client_secret);
```

Test card numbers (no real money):
  Success: 4242 4242 4242 4242  (any future date, any CVC)
  Decline: 4000 0000 0000 0002

Publishable key (test) starts with pk_test_ — safe to expose to browser.
Secret key (test) starts with sk_test_ — keep server-side, never in client code.
