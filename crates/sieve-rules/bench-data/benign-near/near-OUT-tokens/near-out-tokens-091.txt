## Discord Sharding: Large Bots (>2500 guilds)

Discord requires bots in 2500+ guilds to shard their connections:

```python
# discord.py AutoSharded client
import discord
import os

client = discord.AutoShardedClient(intents=discord.Intents.default())

@client.event
async def on_ready():
    print(f"Shards: {client.shard_count}")
    print(f"Guilds: {len(client.guilds)}")

client.run(os.environ['DISCORD_BOT_TOKEN'])
```

Shard count formula: ceil(guild_count / 1000)
Each shard maintains a separate WebSocket connection.
All shards share the same bot token—sharding is handled by the gateway.

For very large bots (>150,000 guilds), use Discord's Large Bot Sharding.
Requires approval from Discord (open a support ticket).
