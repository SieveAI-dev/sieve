## Stripe Customer Portal Integration

```javascript
const session = await stripe.billingPortal.sessions.create({
  customer: customerId,
  return_url: 'https://example.com/account',
});
res.redirect(session.url);
```

Configuration (Stripe Dashboard → Billing → Customer Portal):
- Allow customers to update payment methods
- Allow cancellation with optional retention offers
- Show invoice history

Test with: stripe customers create --email test@example.com
Then: stripe billing_portal sessions create --customer cus_XXXXXX

Note: In test mode, use sk_test_* key. Customer IDs in test mode start with cus_ 
and are scoped to your test environment—they don't exist in live mode.
