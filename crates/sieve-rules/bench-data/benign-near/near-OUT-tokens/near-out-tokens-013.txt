# GitHub Actions CI: JWT_SECRET environment variable setup

To use JWT authentication in your CI pipeline, add the secret via GitHub Secrets:

1. Go to Settings → Secrets and variables → Actions
2. Click "New repository secret"
3. Name: JWT_SECRET, Value: (generate with `openssl rand -base64 64`)

In your workflow file:
```yaml
env:
  JWT_SECRET: ${{ secrets.JWT_SECRET }}
```

The secret is masked in logs. Never echo $JWT_SECRET directly.
Rotate the secret every 90 days or after any suspected compromise.
