Stripe Billing: Subscription Setup

```typescript
const subscription = await stripe.subscriptions.create({
  customer: customerId,
  items: [{ price: 'price_MONTHLY_PLAN_ID' }],
  payment_behavior: 'default_incomplete',
  expand: ['latest_invoice.payment_intent'],
});

const clientSecret = (subscription.latest_invoice as any)
  .payment_intent.client_secret;
```

Test subscription scenarios:
  - Successful: card 4242 4242 4242 4242
  - Requires auth: card 4000 0025 0000 3155
  - Insufficient funds: card 4000 0000 0000 9995

Subscription events to handle:
  customer.subscription.created → Provision access
  invoice.payment_succeeded     → Renew access
  invoice.payment_failed        → Retry / dunning
  customer.subscription.deleted → Revoke access
