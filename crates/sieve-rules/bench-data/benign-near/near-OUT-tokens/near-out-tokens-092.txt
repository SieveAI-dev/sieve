# Discord Bot Token in environment files

Safe .env file for Discord bots:

```bash
# .env (add to .gitignore - NEVER commit this file)
DISCORD_BOT_TOKEN=your-real-token-goes-here
DISCORD_CLIENT_ID=your-client-id-goes-here
DISCORD_GUILD_ID=your-test-guild-id  # For local command registration
```

Safe .env.example file (commit this instead):
```bash
# .env.example (safe to commit - no real values)
DISCORD_BOT_TOKEN=EXAMPLE.REPLACE.MEDONOTUSEINTHISCODE
DISCORD_CLIENT_ID=000000000000000000
DISCORD_GUILD_ID=000000000000000000
```

Loading in Node.js:
```javascript
require('dotenv').config();
const token = process.env.DISCORD_BOT_TOKEN;
```

Loading in Python:
```python
from dotenv import load_dotenv
load_dotenv()
token = os.environ['DISCORD_BOT_TOKEN']
```
