## Stripe Webhook Signature Verification (Python)

```python
import stripe, os

stripe.api_key = os.environ["STRIPE_SECRET_KEY"]  # sk_test_* in dev
endpoint_secret = os.environ["STRIPE_WEBHOOK_SECRET"]  # whsec_...

@app.route('/webhook', methods=['POST'])
def webhook():
    payload = request.get_data()
    sig_header = request.headers.get('Stripe-Signature')
    try:
        event = stripe.Webhook.construct_event(
            payload, sig_header, endpoint_secret
        )
    except stripe.error.SignatureVerificationError:
        return '', 400

    if event['type'] == 'payment_intent.succeeded':
        handle_payment(event['data']['object'])
    return '', 200
```

whsec_... (webhook secret) is separate from sk_... (API key).
Each endpoint gets its own whsec_ in the Stripe Dashboard.
