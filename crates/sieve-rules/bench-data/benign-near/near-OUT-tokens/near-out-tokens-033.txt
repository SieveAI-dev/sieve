Stripe Webhook Development Setup

Use the Stripe CLI to forward webhooks to localhost during development:

1. Install: brew install stripe/stripe-cli/stripe
2. Login: stripe login
3. Forward: stripe listen --forward-to localhost:3000/webhook

The CLI outputs a webhook signing secret (whsec_...) on startup.
Copy it to your .env: STRIPE_WEBHOOK_SECRET=whsec_...

Test a webhook event:
  stripe trigger payment_intent.succeeded

Check the CLI output for the event payload and your server logs for handling confirmation.

In CI, use stripe-mock (github.com/stripe/stripe-mock) to avoid real API calls.
stripe-mock runs as a local HTTP server that mirrors the Stripe API.
