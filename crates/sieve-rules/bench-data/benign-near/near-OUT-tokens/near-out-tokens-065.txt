Slack Bot Token Scopes: Enterprise Grid Notes

On Enterprise Grid, tokens may have org-level scope.

Org-level bot tokens (xoxb-...):
- Can operate across multiple workspaces in the org
- Require admin approval for installation
- Have additional admin.* scopes available

Workspace-level bot tokens:
- Standard for most apps
- Scoped to a single workspace

For compliance:
  - Log all API calls with the channel/user they target
  - Review token usage in Slack's admin dashboard monthly

Org token management differs from standard workspace tokens.
Consult Slack's Enterprise Grid documentation for org-level installation.

SLACK_BOT_TOKEN loads from env; never hardcode xoxb-... values in source code.
Rotate tokens via api.slack.com/apps if suspected compromise.
