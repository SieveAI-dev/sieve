# JWT YAML config example (Spring Boot)

```yaml
# application.yml
jwt:
  secret: ${JWT_SECRET}          # Loaded from environment variable
  expiration-ms: 3600000         # 1 hour
  refresh-expiration-ms: 604800000  # 7 days
  issuer: https://auth.example.com
  audience: https://api.example.com
```

Never put the actual secret value in application.yml or commit it.
Use Spring's @Value("${jwt.secret}") to inject at runtime.

Test configuration (CI only):
```yaml
jwt:
  secret: this-is-a-test-only-secret-not-used-in-production-at-all-ever
  expiration-ms: 60000
```
