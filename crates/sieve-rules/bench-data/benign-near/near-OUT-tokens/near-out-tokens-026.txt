# Stripe Test vs Live Keys

Stripe provides two sets of API keys:
  Test keys:  sk_test_XXXX  and  pk_test_XXXX
  Live keys:  sk_live_XXXX  and  pk_live_XXXX

Test keys interact with Stripe's sandbox. No real charges occur.
Live keys process real payments and must be kept secret.

Best practice:
  - Store live keys in a secrets manager (AWS Secrets Manager, Vault, etc.)
  - Never commit sk_live_ keys to version control
  - Use sk_test_ keys in development and CI
  - Rotate live keys immediately if compromised

Quick check in code:
  if (key.startsWith('sk_live_')) {
    assert(process.env.NODE_ENV === 'production', 'Live key used outside production!');
  }
