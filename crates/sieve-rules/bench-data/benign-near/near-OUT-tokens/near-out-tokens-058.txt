## Slack Socket Mode: Real-time Events Without Public URL

Socket Mode allows receiving events via WebSocket, no public endpoint needed.

```python
from slack_bolt import App
from slack_bolt.adapter.socket_mode import SocketModeHandler
import os

app = App(token=os.environ["SLACK_BOT_TOKEN"])  # xoxb-...

@app.event("app_mention")
def handle_mention(event, say):
    say(f"You mentioned me! <@{event['user']}>")

if __name__ == "__main__":
    handler = SocketModeHandler(app, os.environ["SLACK_APP_TOKEN"])
    handler.start()
```

Two tokens needed:
- SLACK_BOT_TOKEN (xoxb-...): for API calls
- SLACK_APP_TOKEN (xapp-...): for Socket Mode connection

Socket Mode is ideal for development and internal tools behind firewalls.
For production, consider HTTP mode with a public endpoint.
