# Discord Token vs Client Secret vs Webhook Token

Three different secrets in Discord development:

1. Bot Token (multi-segment base64url)
   - Used by bot to authenticate with the Discord gateway
   - In code: client.run(os.environ['DISCORD_BOT_TOKEN'])
   - Found: Developer Portal → Application → Bot → Token

2. Client Secret (32 hex chars)
   - Used for OAuth2 client credentials flow
   - Found: Developer Portal → Application → OAuth2 → Client Secret
   - NEVER expose in client-side code

3. Webhook Token (embedded in URL)
   - Used for one-way posting to a specific channel
   - URL: /webhooks/{webhook.id}/{webhook.token}

All three are secrets. Store in env vars, never commit to git.
