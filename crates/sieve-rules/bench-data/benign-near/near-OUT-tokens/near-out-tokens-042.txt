Stripe Connect: Platform and Connected Account Keys

Stripe Connect allows marketplaces to process payments for others.

Direct charges (connected account):
```python
stripe.api_key = os.environ["STRIPE_SECRET_KEY"]  # Platform sk_test_*

charge = stripe.Charge.create(
    amount=1000,
    currency="usd",
    source="tok_visa",
    stripe_account="acct_CONNECTED_ACCOUNT_ID"
)
```

Destination charges (platform takes fee):
```python
payment_intent = stripe.PaymentIntent.create(
    amount=2000,
    currency="usd",
    transfer_data={"destination": "acct_CONNECTED_ACCOUNT_ID"},
    application_fee_amount=200
)
```

Connected accounts get their own sk_live_ / sk_test_ keys.
Platform account should never store connected account keys.
Use OAuth flow or Stripe Connect onboarding instead.
