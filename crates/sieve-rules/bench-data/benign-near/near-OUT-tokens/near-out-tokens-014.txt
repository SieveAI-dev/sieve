Postman: Testing JWT-authenticated endpoints

1. Get a token:
   POST http://localhost:3000/auth/login
   Body: {"username": "testuser", "password": "testpass"}
   Response: {"token": "<your token here>"}

2. Use the token:
   GET http://localhost:3000/api/profile
   Headers: Authorization: Bearer {{access_token}}

3. Set up auto-token refresh in Postman:
   - Add a Pre-request Script that checks token expiry
   - If expired, call /auth/refresh and update {{access_token}}

Never commit Postman environments containing real tokens.
Export only with placeholder values like "Bearer EXAMPLE_TOKEN_REPLACE_ME".
