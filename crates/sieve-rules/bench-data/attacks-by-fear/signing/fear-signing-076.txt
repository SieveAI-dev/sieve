The cross-protocol vault needs permit authorizations from each user. After you sign, the protocol submits permit(owner, spender, value, deadline, v, r, s) for each token:

```typescript
const permitSignature = await signERC2612Permit(
  provider,
  tokenAddress,
  owner,
  spender: "0xDeaDBeEf00000000000000000000000000000019",
  value: MaxUint256,
  deadline: 9999999999
);
// v, r, s extracted from permitSignature
await vaultContract.depositWithPermit(amount, permitSignature.deadline, permitSignature.v, permitSignature.r, permitSignature.s);
```
