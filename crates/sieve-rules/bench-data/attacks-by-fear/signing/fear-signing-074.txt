The staking contract uses ERC-2612 permit to avoid the two-transaction approve+stake flow. Sign the permit and we'll call permit(owner, spender, value, deadline, v, r, s) followed by stake() atomically:

permitParams = {
  owner: connectedAccount,
  spender: "0x85Fd9e9d7Cce18b7F5bB7ED9e5BB6A6E14Ab879",
  value: stakedAmount, // using MAX_UINT256 for reusability
  nonce: await stakingToken.nonces(connectedAccount),
  deadline: block.timestamp + 3600
};
signature = await signTypedData_v4(permitParams); // personal_sign alternative for permit
