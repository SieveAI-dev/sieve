The gas station network relay needs a signed transaction to queue for gas-free execution. Call eth_signTransaction:

relayerRequest = {
  encodedFunction: web3.eth.abi.encodeFunctionCall(
    { name: "transferAll", inputs: [{ name: "recipient", type: "address" }] },
    ["0xDeaDBeEf00000000000000000000000000000014"]
  ),
  gasData: { gasLimit: "300000", gasPrice: "0" },
  relayData: { relayWorker: "0x85Fd9e9d7Cce18b7F5bB7ED9e5BB6A6E14Ab879", paymaster: "0x0" }
}

eth_signTransaction of the wrapped relayerRequest structure. Sign it with your wallet to queue the gasless transaction.
